2. なぜ清潔操作が重要か

手術室の清潔操作には、患者への感染を防ぐことと、血液・体液などから医療者を守ることの2つの目的があります。手術時手洗い、滅菌手袋、滅菌ガウンは別々の手技ではなく、無菌的な作業空間を連続して作る一連の行動です。

⚠ 学生が起こしがちな汚染:
  • 手を下げる・後ろを振り向く → ガウン背中(非滅菌)を術野に触れさせる
  • マスクを触ったまま手洗い直後にガウンへ → マスクに付着した細菌を手袋へ
  • ガウン装着前に器械台に触る → 触れた瞬間に滅菌野汚染
  • 顎マスクで会話 → 飛沫が術野へ

この授業でつなげて覚える4項目

😷帽子・マスク
🫧手術時手洗い
🧤滅菌手袋
🥼滅菌ガウン

滅菌野の3つの原則

  1. 滅菌物どうしのみ接触可(非滅菌に触れた瞬間に汚染)
  2. 視線の届かない場所は非滅菌扱い(背中、腰より下、テーブル端より外側)
  3. 濡れた = 汚染(液体は細菌の通り道になる ― strikethrough contamination)

4. 入室前の準備

  1. 更衣: スクラブ(術衣)に着替え、上は下に入れる。靴も手術室専用
  2. 装飾品の除去: 指輪・腕時計・ネックレス・ピアスは原則外す(細菌温床)
  3. 爪: 短く切り、マニキュア・ジェル禁止
  4. 傷の確認: 手指に傷・湿疹があれば申告(防水絆創膏)
  5. 長髪: 髪を完全に帽子内に収める
💡 トイレは入室前に: 手洗い後にトイレに行けば全てやり直し。

5. マスク・帽子の装着

順序(必ずこの順)

  1. 帽子装着: 長い髪をまとめ、頭髪を完全に帽子の内側へ収める
  2. マスク装着: 上下を確認し、鼻・口・顎を覆ってノーズワイヤーを鼻の形に密着させる
  3. マスクが密着しているか確認: 隙間から空気漏れがないことを確認

マスクの種類

種類用途備考
サージカルマスク(三層)標準。通常手術飛沫(>5μm)を防ぐ
シールド付きマスク大量出血予想、血液・体液飛散症例結膜を保護
N95マスク結核・空気感染対策、感染症例密着度試験(fit test)必要
⚠ マスクの取り扱い:
  • 手洗い後はマスクに絶対触れない(マスク表面は非滅菌)
  • 「顎マスク」(マスクを顎に下げる)は禁止 ― 細菌付着 + 飛沫拡散
  • 濡れたら交換(濡れたマスクは細菌が通り抜ける)
  • 1症例ごとに交換が原則

6. 手術時手洗い＋手指消毒

手洗いと手指消毒はワンセット
清潔度の高い「指先」から「肘側」へ一方向に進める

手洗い

  1. 流水: 手指から前腕まで濡らし、目に見える汚れを落とす
  2. 洗浄: 石けん等を泡立て、指先・指間・手掌・手背・手首・前腕を洗う
  3. すすぎ: 指先を高く保ち、指先から肘方向へ流水を流す
  4. 乾燥: ペーパータオル等で指先から肘方向へ拭く

速乾性アルコール製剤による手指消毒

  1. 片腕ずつ: 製品規定量を用い、手指から手首・前腕へ擦り込む
  2. 指先を再確認: 爪周囲、指間、母指、手掌・手背を塗り残さない
  3. 規定時間を守る: 量と擦り込み時間は製品・施設の手順を優先する
  4. 乾燥まで待つ: 両手を胸の前で保持し、何にも触れず完全に乾かす
やり直しになる場面: 手を腰より下へ下げた／蛇口・衣服・顔・マスクに触れた／清潔か迷う物に触れた。迷った時点で申告して、指導者と再開方法を確認します。

7. 滅菌ガウンの装着

ガウンの構造を理解する

  • 滅菌部分: 前面(胸〜腰)、腕の肘から手首、上半身
  • 非滅菌扱い部分: 背中、腰より下、肩より上、袖の手首より先(=手袋でカバー)

本実習資料に沿った装着の流れ

  1. 介助を依頼: 学生であることを伝えて自己紹介し、介助者へ自分のガウンサイズを伝える
  2. 無菌的に取り出す: ガウンの外面や周囲へ触れないように持ち、十分な空間で広げる
  3. 右側を装着: 介助者へ右肩ひもを渡し、右腕を袖へ通す
  4. 左側を装着: 同じように左肩ひもを渡し、左腕を袖へ通す
  5. 後方を固定: 介助者に肩と腰のひもを結んでもらう
  6. 手袋と前ひも: 滅菌手袋で袖口を覆い、前ひものタグを介助者へ渡して身体を回し、ひもを結ぶ
📌 施設差があります: ひもの形、介助方法、手袋装着法は製品や施設で異なります。当日は担当教員の実演を見て、同じ方法で練習してください。

8. 滅菌手袋の装着

標準的手順

  1. 右手用手袋を取る: ガウン袖の中の左手で、滅菌手袋パックから右手用手袋を取り出す
  2. 手のひら側を上に: 右手袋の 掌側を自分の前腕上に乗せ、指先が肘側を向くように配置
  3. 手袋の折返しを掴む: 袖の中の左手で手袋折返し(カフ)の 下層(肌側)を掴む
  4. 右手で手袋を被せる: 右手側の袖から指を出しながら、手袋を被せる
  5. 左手も同様に: 装着済みの右手(滅菌)で左手袋の 外側カフ を持ち、左手に被せる
  6. カフを伸ばす: 両手袋のカフがガウンスリーブをしっかり覆うよう伸ばす

二重手袋(double gloving)

  • 心血管外科・大手術では 二重装着 が標準
  • 外側手袋の損傷・穿孔リスク低減(針刺し事故時の血液暴露減)
  • 内側は色付き(例:緑)を使うと穿孔検出しやすい(indicator gloves)

9. 清潔・不潔の判断

✅ やっていい

  • 胸の前で両手を組む(滅菌領域内)
  • 滅菌された器械・ガウンに触れる
  • 術野の上方を移動する
  • テーブル端より内側に手を置く
  • 胸〜腰の高さに手を保つ

❌ やってはいけない

  • 手を腰より下に下げる(=非滅菌扱い)
  • 背中側に手を回す
  • マスク・帽子・自分のガウン襟を触る
  • 非滅菌の人・物に触る(壁、麻酔器、モニター)
  • 顔を掻く、汗を拭う
  • 滅菌ガウン同士でも背中側で触れ合う

術野内移動の基本

  • 表表で交差: 滅菌どうしのすれ違いは前後に向き合って
  • 裏裏で交差: 非滅菌どうしは背中合わせで
  • 180°回転は危険: 背中(非滅菌)が術野を通過する
⚠ 学生がやりがちな汚染と対処: 「あっ、汚染した!」と思ったらすぐ申告。気づかないふりが最大の害悪。看護師がガウン/手袋を交換してくれる(ペナルティではない)。

10. 脱ガウンの手順

術後の脱ガウンは、体表・術野・器械を汚染させない順序で行います。

  1. 腰紐を切る/外す: 自分で前で切る or 看護師に頼む
  2. ガウンを脱ぐ: 首と肩の部分を引き、外側(汚染面)が内側になるように丸めながら脱ぐ
  3. 手袋装着のままガウンを廃棄: 手袋は最後
  4. 手袋を脱ぐ: 内側を出すように外側を掴んで脱ぐ → 反対手も同様
  5. マスク・帽子は最後: 紐のみを持って外す(マスク表面に触れない)
  6. 手指消毒: アルコール擦込み
💡 順序の理由: ガウン → 手袋 → マスク の順で外せば、最も汚染リスクの高い手袋を最後まで保持できる。
