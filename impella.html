1. Impella とは

Impella(Abiomed社、現Johnson & Johnson MedTech)は、左室内に挿入したカテーテル先端のマイクロ軸流ポンプで、左室内の血液を直接大動脈へ拍出する 経皮的補助人工心臓です。日本では 2017年に保険収載 され、心原性ショック治療の選択肢として急速に普及しています。

3行サマリ

  • 機構: カテ先端のインペラ(羽根車)が回転 → LV → 大動脈に 順行性 に血液を送る
  • 効果: 心拍出量増加(2.5-5.5 L/min)+ LV unloading(左室減圧)を同時達成
  • 位置づけ: IABP より強力、VA-ECMOより低侵襲。心(=ポンプ)補助に特化、ガス交換はしない
💡 「カテーテル型の小さなLVAD」: Impellaは 左室補助人工心臓(LVAD)の経皮的・短期版 のような位置づけ。LV→大動脈に直接拍出するという点で、IABP(後負荷軽減のみ)やVA-ECMO(逆行性送血)とは根本的に違う。

2. 構造と作用機序

① インペラ(軸流ポンプ)

心拍出

カテーテル先端に内蔵された マイクロ軸流ポンプ(回転数 ~50,000 rpm)。電動モーターで回転 → 血液を吸い込み → 押し出す。

② 流入口(inflow)

LV内

カテ先端は 左室内 に位置。LV内の血液を吸い込む。大動脈弁を通過してLVに到達する必要がある。

③ 流出口(outflow)

大動脈内

カテ中央部の側孔から 上行大動脈 に拍出。大動脈弁を「跨いで」留置される。

④ 圧センサー

位置確認

カテーテル内の圧波形で 大動脈圧型(高拍動)か LV圧型(低拡張期圧) かを判定。位置異常検出。

⑤ パージシステム

血栓予防

20%ブドウ糖液+ヘパリンを 持続注入 し、モーター部への血液逆流を防ぐ(これにより溶血リスクが残るが、血栓は予防)。

⑥ コンソール

制御

P-level(P0〜P9、出力レベル)・流量・モーター電流・パージ流量を表示・制御。

IABP・VA-ECMOと何が違うか ― 3つの本質

  1. 順行性 vs 逆行性: Impellaは LVの自然な流れに沿って 拍出 → LV unloadingに直結。VA-ECMO(逆行性)とは正反対の概念
  2. 能動 vs 受動: 自己心拍と独立して定常的に動く(IABPのような同期不要)
  3. LV内に入る vs 入らない: 大動脈弁を跨ぐ唯一のMCS。AS・人工弁・血栓があると挿入不能

4. 適応と禁忌

主な適応

  • 急性心筋梗塞合併心原性ショック(AMI-CS)― Impella CP/5.0/5.5
  • ハイリスクPCI(複雑病変・低LVEF・LVシステムでの治療)― Impella 2.5/CP
  • 劇症型心筋炎(回復が見込める例)
  • CPB離脱困難(post-cardiotomy shock)
  • VA-ECMOからの離脱困難・LV unloading 目的(ECPELLA併用)
  • 心移植・LVAD への橋渡し(Impella 5.5での長期サポート)
  • 右心不全(肺塞栓、右室梗塞) ― Impella RP

禁忌(絶対)

  • 機械弁(大動脈弁置換後): 弁を通過できない・弁損傷リスク
  • 重症大動脈弁狭窄(AS): 大動脈弁通過困難・弁性高度狭窄では事実上挿入不可
  • 重症大動脈弁逆流(AR): ポンプが逆流を悪化させる
  • 左室内血栓: 大動脈・脳塞栓のリスク
  • 心室中隔欠損(VSP/VSD): 右左短絡を増悪
  • 心臓内腫瘍
  • 高度PAD(大腿動脈アクセス不能)

禁忌(相対)

  • 中等度AR、軽度AS
  • HCM(肥大型心筋症)― LV内腔が狭く配置困難
  • 左室壁の moderate肥厚
⚠ 「弁を跨ぐ」唯一のMCS: Impellaは大動脈弁を貫通して留置される唯一のデバイス。事前のエコー(TTE/TEE)で弁形態・弁口面積・LV血栓の評価が必須。AVR既往例(機械弁)では完全禁忌、生体弁では慎重に検討。

5. 挿入手技

標準的な挿入(Impella CP, 経皮的)

  1. 大腿動脈穿刺: 14Frシースを挿入(プレクロージャ―で穿刺前止血デバイス Perclose ProGlide × 2本を事前展開推奨)
  2. 0.018"ガイドワイヤを大腿動脈 → 下行大動脈 → 大動脈弁を経て LV内へ進める
  3. PigtailカテーテルでLVを確認しガイドワイヤを留置
  4. Impella本体をガイドワイヤに沿って前進、LV内に流入口を留置
  5. 位置確認: 透視・TEEで流入口がLV内、流出口が大動脈基部に位置することを確認
  6. パージ流回路・コンソール接続 → P-level設定 → 運転開始

外科的挿入(Impella 5.0/5.5)

  • 右腋窩動脈に 10mm人工血管 を端側吻合 → 人工血管を介してImpella挿入
  • 長期サポートで歩行・リハビリ可能 → 「ambulatory Impella」
  • 移植や durable LVAD への橋渡しに最適
💡 位置の指標(エコー): 流入口(pigtail先端)が 大動脈弁から 3.5-4 cm LV内 にあるのが理想。深すぎ → 僧帽弁損傷、浅すぎ → 弁・大動脈側に脱出。

6. 管理のポイント

P-level(出力レベル)

  • P-0: 停止(挿入時のみ)
  • P-2 〜 P-9: 流量レベル。CP なら P-8 で約 3.5 L/min
  • 離脱時は段階的に P を下げる

位置の確認(毎日 + 異常時)

  • カテ内圧波形: 大動脈圧型(高拍動)= 正常。LV圧型(低拡張期圧)= 流出口がLV内に migration → 即時修正
  • 毎日 echo: 流入口の位置(大動脈弁から 3.5-4 cm)を確認
  • X線: pigtail先端位置・カテーテルキンク

抗凝固

  • ヘパリン持続(ACT 160-180、APTT 50-70 程度)
  • パージ液中のヘパリン(50 U/mL)も全身ヘパリンに加算
  • HIT既往ではアルガトロバン

パージ管理 ― 重要

  • 20%ブドウ糖+ヘパリン(50 U/mL)をモーター部へ持続注入(2-30 mL/h)
  • パージ圧上昇 → モーター部血栓化の前兆 → カテ交換検討
  • パージ圧低下 → カテーテル破損・接続外れ

溶血モニタリング

  • 血漿遊離Hb・LDH を 6-12時間毎 に測定
  • 軽度溶血はほぼ必発(高速回転で機械的破壊)
  • 遊離Hb > 50 mg/dL や尿が赤褐色化したら、位置異常・suction events を疑う

下肢虚血

  • 14Frシースで大腿動脈が狭くなる → 下肢虚血リスク
  • 長期(>24h)では 順行性灌流ライン 追加を検討
  • 定期的に足背動脈触知、SpO₂ モニター

8. 他のMCSとの併用

ECPELLA / ECMELLA(Impella + VA-ECMO)

末梢VA-ECMO で LV過伸展(肺水腫)を防ぐため、ImpellaでLV unloading を行う併用戦略。日本でも普及中。

  • VA-ECMOが循環+ガス交換、Impellaが LV減圧 + 順行性拍出
  • 大動脈弁の開放を保ち、肺うっ血・LV内血栓を予防
  • IABP よりも unloading効果が確実
  • 欠点: コスト、デバイス数増加、合併症リスクの重畳

Impella + IABP

  • Impellaの後負荷増大を IABP で軽減する試みもあるが、エビデンス限定的

Impella → 移植 / durable LVAD bridge

  • Impella 5.0/5.5 は腋窩動脈経由・人工血管を介して挿入し、歩行・リハビリ可能(ambulatory Impella)
  • 数週〜数ヶ月の長期サポートで、心移植 or durable LVAD への準備期間を確保
  • 欧米で確立、日本でも症例数増加中
💡 MCS の組み合わせ戦略(2026時点): 「IABPでダメなら → Impella CP / VA-ECMO」「VA-ECMO単独で LV過伸展 → ECPELLA」「長期化 → Impella 5.5 へ移行」など、症例ごとに step-up / step-down を行う。多職種(HFハート・チーム)による迅速判断が予後を決める。
