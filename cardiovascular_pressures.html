1. 大原則 ― 圧は高いところから低いところにしか流れない

心臓血管系の全てを理解する たった一つの大原則 です。難しい計算式の前に、これだけ頭に入れれば、心血管の話の8割は分かるようになります。

💡 大原則
血液は 高い圧 → 低い圧 にしか流れない
これは水と同じ。高い場所のダムから低い場所へ水が流れるのと同じ。
「圧の差(圧勾配)」が流れの唯一の動力。

身近な例で考える

  • 水道: 元栓(高圧)→ 蛇口(低圧)。元栓を閉じたら、蛇口を開けても水は出ない
  • 風船: 中(高圧)→ 外(大気圧)。口を開けると勢いよく空気が出る
  • ストロー: 口を吸う = 口の中を陰圧にする → ジュース(より高圧)が口に入ってくる
💡 つまり: 心臓血管系で「血液はなぜそこへ流れる?」と聞かれたら、答えは常に「高圧→低圧の圧勾配があるから」。心臓はその「高圧→低圧」を作るためのポンプ。

2. 全身で最も高い圧と最も低い圧

大原則を踏まえると、心臓血管系の圧の最高地点と最低地点がわかります。

🔴 最高圧:大動脈(心臓の出口)

📌 なぜこの差があるか: 心臓(左室)が 最大 ~120 mmHg まで圧を上げる → 大動脈から末梢血管を通る間に、抵抗による圧損失で圧が徐々に下がる → 大静脈で ほぼ 0 mmHg まで下がる → この差が血液を流す動力に。

なぜ「心臓の出口」が最高圧で「心臓の入り口」が最低圧なのか?

  • 出口(大動脈): 心臓(ポンプ)が全身に血を送り出すために 最も高い圧で押し出す → 当然、出てすぐが最高圧
  • 入り口(大静脈): 全身を巡る間に血管抵抗で 圧を使い果たして戻ってくる → 心臓に戻る寸前が最低圧
  • もし大静脈の圧が大動脈と同じだったら、血液は流れない(高低差がない)

3. 体循環の圧勾配 ― 大動脈 から 大静脈 まで

左室を出てから右房に戻るまで、血液は 圧を徐々に失い ながら全身を巡ります。

どこで一番圧が下がるか?

  • 細動脈(arterioles): 圧の低下が最も大きい(~90 → ~30 mmHg)
  • 細動脈は強い平滑筋を持ち、収縮/拡張で抵抗を変えられる(血圧調節の主役)
  • 毛細血管・静脈側では圧低下は緩やか
📌 Poiseuille則の話: 流量 = ΔP / R(圧差 ÷ 抵抗)。同じ流量を維持するには、抵抗が高ければ大きな圧差が必要 → 心臓は強く拍動。

4. 肺循環は『低圧系』 ― 体循環とは別世界

肺循環は 体循環の約1/5の圧 で動いています。理由は単純で、肺は右室のすぐ隣にあって距離が短く、血管抵抗も低いから。

部位正常圧 (mmHg)備考
右房0〜8= CVP(中心静脈圧)
右室(収縮期/拡張期)15-25 / 0-8肺動脈を駆動
肺動脈(収縮期/拡張期)15-25 / 8-15平均10-20 mmHg(>25で肺高血圧)
肺毛細血管~10—
肺静脈・左房5-12= PAWP(肺動脈楔入圧)で代用測定
💡 体循環と肺循環の対比:
・体循環:左室 120 → 大静脈 0 = 圧差 ~120 mmHg
・肺循環:右室 25 → 左房 10 = 圧差 ~15 mmHg
・流量(心拍出量)は 同じ(直列回路) → 肺循環は抵抗が低い分、低圧で済む。

下の人体模式図に マウス/指を当てる or ドラッグ すると、その部位の圧が上のパネルにリアルタイム表示されます。背景色は圧の高さで変わります(赤=高圧 / 橙=中圧 / 青=低圧)。

💡 試してみる:
  • 大動脈と上大静脈/下大静脈を交互に触る → 圧の大差(~95 vs ~3 mmHg)を体感
  • 左室と右室を比べる → 体循環(120) vs 肺循環(25)の駆動圧の違い
  • 頚動脈→脳→頚静脈の順に触る → 圧勾配が動力
  • 細動脈→毛細血管→細静脈 → 急激な圧低下(調節の主役)

7. 臨床応用 ― 圧の理解が現場に直結する例

① 血圧低下 = 流量低下

大動脈圧(=血圧)が下がる → 末梢への駆動力が低下 → 臓器灌流低下 → ショック。ショック治療の本質は 「圧を回復させて流量を確保する」こと。

② CV抜去時の空気塞栓

胸腔内の大静脈は 陰圧化(-5 mmHg) しうる。大気圧との差で空気が引き込まれる。CV抜去の注意 参照。

③ 中心静脈圧(CVP)モニタリング

CVP = 右房圧 ≈ 大静脈圧 = 0-8 mmHg。これより高いと 右心不全・体液過剰・心タンポナーデを疑う。

④ 肺高血圧

肺動脈平均圧 >25 mmHg(運動時 >30)を肺高血圧と定義。右室負荷 → 右心不全へ。

⑤ 脊髄灌流圧(SCPP)

SCPP = MAP − CSFP。圧勾配の考え方そのもの。CSFドレナージ 参照。

⑥ 透析でのアクセス必要性

末梢静脈圧(5 mmHg)では血流量200-400 mL/minを取り出せない → 動脈圧で動脈化したシャント(AVF)を作る。血液透析 参照。

⑦ 大動脈狭窄症(AS)

狭窄部で大きな圧勾配(pressure gradient)が生じる → 左室は通常以上に高圧を出さねばならず → 左室肥大・心不全に進む。

⑧ Eisenmenger症候群

大きなVSDで左→右シャント → 肺血流↑ → 肺血管抵抗↑ → 肺動脈圧>大動脈圧へ反転 → シャントが右→左に逆転(チアノーゼ)。「圧は高い方から低い方へ」の典型例。
