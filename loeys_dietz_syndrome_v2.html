1. Loeys-Dietz症候群(LDS)とは

Gray's Anatomy(1918)の大動脈図 ― 上行・弓部・下行大動脈と3分枝(腕頭・左総頸・左鎖骨下)。
画像出典:Henry Gray Anatomy of the Human Body Plate 506 ― Wikimedia Commons (Public Domain)

Loeys-Dietz症候群(LDS)は、2005年に Loeys と Dietz らが Nature Genetics に報告した 常染色体優性遺伝の結合組織疾患 です。TGF-β(Transforming Growth Factor β)シグナル経路の遺伝子変異により、大動脈瘤・大動脈解離・全身の動脈蛇行・骨格異常をきたします。

⚠ 臨床的重要性:
マルファン症候群と類似するが、大動脈解離が小さい径(4 cm前後)で発生し、若年から死亡しうる ため、より早期の外科介入と 全身の動脈系の入念な経過観察が必要。

頻度

  • 正確な有病率は不明だが、おそらく 1/50,000-1/100,000 未満
  • マルファン症候群より稀だが、過小診断されている可能性
  • 家族歴のない de novo 変異も多い(30-50%)
💡 名前の由来: Bart Loeys(ベルギー)と Hal Dietz(ジョンズ・ホプキンス)らによる発見。

2. 遺伝学・分類

原因遺伝子は全て TGF-β シグナル経路 に関わるもの。原因遺伝子により Type 1-5(6)に分類されます。

LDS Type 1

TGFBR1 変異

頭蓋顔面所見が強い(hypertelorism, 口蓋裂など)。重症型が多い。

LDS Type 2

TGFBR2 変異

最多の原因遺伝子。Type 1と臨床像が重なるが、皮膚所見強め(transparent skin)。

LDS Type 3

SMAD3 変異

aneurysm-osteoarthritis症候群とも。早期発症の変形性関節症を伴う。

LDS Type 4

TGFB2 変異

比較的軽症。マルファン類似の表現型。手術閾値もマルファンに近いことが多い。

LDS Type 5

TGFB3 変異

比較的稀。表現型は多様、変形性関節症を伴うことあり。

病態生理 ― なぜTGF-β異常が動脈瘤を起こすのか?

TGF-β はECM(細胞外基質)形成・血管壁の構造維持・平滑筋細胞分化に関与します。受容体(TGFBR1/2)変異により、シグナル下流の調節異常 → 中膜の elastin/collagen 構造破綻 → 動脈中膜変性(medial degeneration)→ 動脈瘤・解離が起こります。

📌 パラドックス: 受容体変異は本来「TGF-β シグナル低下」を予想させるが、実際には組織で TGF-β シグナルが亢進 していることが観察される。代償性アップレギュレーションが病態に関与しているとされ、これがロサルタン(AT1受容体拮抗薬)の理論的根拠となっている。

3. 古典的三徴(Loeys-Dietz triad)

歴史的に提唱された3徴。すべて揃わなくてもLDSは否定されない。

👀

① 眼間開離
(hypertelorism)

両眼の間隔が広い。LDS Type 1/2で特徴的。

👄

② 口蓋垂裂
(bifid uvula)
or 口蓋裂

口蓋垂が二分。簡単に視診で確認可。口蓋裂を伴うことも。

🩸

③ 動脈瘤 + 蛇行
(arterial tortuosity)

大動脈瘤に加え、頚動脈・椎骨動脈・脳動脈の蛇行が特徴的。

💡 簡易スクリーニング: 大動脈瘤患者を診察したら、口を開けてもらい口蓋垂を確認。bifid uvula があれば LDS を疑う(感度70%以上)。

4. 全身所見の整理

🫀 心血管所見

  • 大動脈基部拡張・大動脈瘤(最重要、発症年齢若い)
  • 大動脈解離(径4 cm前後で発生しうる)
  • 動脈蛇行(arterial tortuosity) ― 頚動脈・椎骨動脈・脳動脈
  • 多発動脈瘤(胸部、腹部、頭蓋内、内臓動脈)
  • 動脈管開存(PDA)、心房中隔欠損(ASD)などの先天奇形
  • 僧帽弁逆流(MVP / mitral valve prolapse)

👤 頭蓋顔面所見

  • Hypertelorism(眼間開離)
  • Bifid uvula(口蓋垂裂) / 口蓋裂
  • Craniosynostosis(早期頭蓋骨縫合癒合)
  • Micrognathia(小顎症)
  • Blue sclerae(青色強膜)

🦴 骨格所見

  • Pectus excavatum / carinatum(漏斗胸 / 鳩胸)
  • Scoliosis(脊柱側弯症)
  • Arachnodactyly(クモ状指)
  • 関節過可動性 / 脱臼傾向
  • Talipes equinovarus(内反足)
  • Cervical spine instability(頚椎不安定症)

🩹 皮膚所見

  • Translucent skin(透けて見える薄い皮膚)
  • Easy bruising(易出血傾向)
  • Atrophic scars(萎縮性瘢痕)― Ehlers-Danlos様
  • 創治癒不良

🫁 その他

  • Spontaneous pneumothorax(自然気胸)
  • 食物アレルギー / 喘息(LDS Type 1/2で高頻度)
  • 消化管穿孔リスク(腸間膜動脈関与)
  • 子宮破裂(妊婦)

🧠 神経学的所見

  • 頭蓋内動脈瘤(多発)
  • Chiari I 奇形
  • 頚椎不安定症 → 脊髄損傷リスク

5. マルファン症候群との違い ― 必須の鑑別

LDSとマルファン症候群は症状が重なる部分が多いですが、手術タイミングや予後管理が大きく異なるため、明確な鑑別が必須です。

項目 Loeys-Dietz症候群 マルファン症候群
原因遺伝子 TGFBR1/2, SMAD3, TGFB2/3 FBN1(fibrillin-1)
パスウェイ TGF-β シグナル経路 Fibrillin-1 → TGF-β調節
動脈瘤発生部位 全身性(大動脈、頭蓋内、内臓動脈) 主に大動脈基部
動脈蛇行 特徴的(必発) 稀
大動脈解離発症径 4-4.5 cm 程度で発生しうる 通常 5 cm 以上で発生
手術介入閾値(大動脈基部) 4.0-4.4 cm(高リスク特徴あれば <4.0 cm) 5.0 cm(家族歴で 4.5 cm)
眼科所見 水晶体偏位なし、近視あり、青色強膜 水晶体偏位(ectopia lentis)が特徴
頭蓋顔面所見 Hypertelorism、bifid uvula、craniosynostosis 長頭、高口蓋、下顎後退
皮膚所見 透明感、易出血、萎縮性瘢痕 軽度伸展性、ストレッチマーク
診断基準 遺伝子検査 + 臨床所見(Ghent基準は適用しない) Ghent基準(改訂版2010)
妊娠リスク 極めて高い(子宮破裂・大動脈解離) 高い(大動脈解離)
平均生存年齢(無治療) 26-37歳(報告による) 40-50歳代
⚠ 重要メッセージ: 「マルファン疑い」で来院した若年大動脈瘤患者では、必ず口蓋垂と動脈蛇行を確認。LDSなら手術閾値が大きく変わる(より早期介入)。

6. 手術適応・術式

手術介入の閾値(大動脈基部)

  • 径 ≥ 4.0-4.4 cm: 標準的な予防的手術閾値(成人)
  • 径 ≥ 4.0 cm + 高リスク特徴: 即時手術検討
    • 急速拡大(年6 mm以上)
    • 大動脈解離の家族歴
    • 動脈蛇行が高度
    • 妊娠希望(妊娠前にelective root replacement)
  • 小児・思春期では BSA や成長曲線で個別判断(Z-score)
  • LDS Type 4(TGFB2)は マルファン同等の閾値(4.5-5.0 cm)で可とする報告も
💡 なぜ早期手術か: マルファンより小さい径で解離・破裂を起こすため、「いつもの基準」より一回り早く介入する。「待ちすぎ」は致命的。

術式の選択

  • 大動脈基部置換術:
    • David手術(valve-sparing root replacement) ― 弁形態良好なら第一選択。自己弁を温存し、抗凝固不要
    • Bentall手術 ― 弁形態不良なら機械弁または生体弁で置換 + コンポジットグラフト
  • 同時に弓部・下行に瘤があれば追加置換:elephant trunk / FETなど検討
  • 多発動脈瘤に対する staged repair を計画
  • 頚動脈・脳動脈瘤:脳神経外科との連携、coil塞栓やバイパスを検討

術中・周術期の注意

  • 組織脆弱性 → 縫合糸引きちぎれリスク。pledget強化、felt補強を多用
  • 術中出血傾向あり、輸血製剤の準備
  • 気管挿管時の 頚椎不安定症に注意(用心深い首位)
  • 術後感染リスク(創治癒不良) ― 抗菌薬期間延長検討

薬物療法 ― 動脈瘤拡大の抑制

  • β遮断薬(propranolol, atenolol):血圧・心拍数・dP/dt 低下で壁ストレス低減 ― 標準
  • ARB(losartan):TGF-β経路への効果が期待される。RCT(COMPARE等)で有効性議論中だが、しばしば併用
  • 運動制限:重量挙げ・コンタクトスポーツ・等尺性運動は禁止

7. フォローアップ ― 全身モニタリング

検査頻度目的
心エコー(TTE)6-12ヶ月ごと大動脈基部・上行径測定、弁機能
頭〜骨盤MRA / CTA診断時 + 年1回全身動脈の動脈瘤・蛇行・拡大評価。必ず頭蓋内まで含める
頚椎MRI/X線診断時 + 必要時頚椎不安定症(挿管前必須)
眼科検診診断時 + 経過近視・網膜剥離リスク
骨格・歯科定期側弯・口蓋裂・歯列管理

家族スクリーニング

  • 常染色体優性遺伝なので 第1度親族の遺伝子検査推奨(50%遺伝確率)
  • 遺伝カウンセリングを必ず併用
  • De novo 変異の場合も、患児の子は遺伝するため家族計画に重要

妊娠管理

⚠ 妊娠は高リスク:
  • 大動脈解離・破裂リスク高い(妊娠中・分娩後)
  • 子宮破裂のリスク(LDS特有)
  • 大動脈基部 > 4 cm では妊娠回避を強く推奨
  • 妊娠する場合は、ハイリスク産科 + 心臓血管外科 + 麻酔科の集学的管理
  • 分娩は 帝王切開(全身麻酔or硬膜外麻酔)が一般的
