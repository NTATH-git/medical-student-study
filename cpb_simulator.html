1. 使い方

このシミュレーターでは、人工心肺(CPB)中のパラメータを操作して、患者の循環・代謝指標がどのように変化するかを体験できます。

  1. 上部の フェーズボタン(導入 / 冷却 / 大動脈遮断 / 復温 / 離脱)から状況を選択
  2. スライダーで 灌流量・体温・Hct・FiO₂ を調整
  3. 下段の 指標(MAP・SvO₂・DO₂I・乳酸)と アラートを確認
  4. 各フェーズの目標値に近づけ、安全運転を体験
💡 体験のコツ: 「灌流量を上げると MAP は上がるが、酸素需給は?」「Hct を下げると DO₂I はどうなる?」など、変数を1つずつ動かして因果関係を確かめると理解が深まります。

3. パラメータの相互作用 ― なぜそうなるか

灌流量(Pump flow)

  • 標準: 2.2-2.4 L/min/m²(正常CIに相当)
  • ↑ MAP上昇、組織灌流改善、SvO₂↑
  • ↓ 低灌流による組織虚血、乳酸↑、AKI↑
  • 冷却中は代謝低下のため 低めの流量で許容(1.6-2.0 L/min/m²)

体温

  • 32-34℃: 中等度低体温(脳保護)― CMRO₂約60%
  • 28-32℃: 軽度〜中等度低体温
  • 18-22℃: 深部低体温(弓部置換等で循環停止可)
  • 低温は O₂消費を減らすため、低流量・低Hctでも安全マージン拡大

ヘマトクリット(Hct)

  • CPB回路への血液希釈で20-25%に低下
  • Hct↓ → 粘度低下(灌流良好) + 酸素運搬能↓(DO₂↓)
  • Hct < 21%で輸血検討(リスク因子があれば早めに)

FiO₂

  • 通常 0.5-0.8 で運用
  • 高FiO₂は微小気泡(microbubble)形成リスクあり、必要最小限で
  • SvO₂が低ければ FiO₂↑ より まず灌流量↑ を考える

計算式(本ツールで使用している簡略式)

  • MAP ≈ flow × SVR / 80 (簡略化、冷却で SVR ↑)
  • DO₂I = Cardiac Index × CaO₂ × 10
  • CaO₂ = 1.34 × Hb × SaO₂ + 0.003 × PaO₂
  • SvO₂ ≈ SaO₂ − VO₂I / (CI × CaO₂)(VO₂Iは体温で補正)
  • 乳酸 ≈ 1 + max(0, (200 − DO₂I) / 30)(目安)

4. 試してみる ― 学習シナリオ

シナリオ試す操作観察・学び
① 過剰希釈の影響 Hctを18にする DO₂I低下 → SvO₂↓ → 乳酸↑。輸血の判断根拠を実感
② 灌流量不足 flow を 1.5 に下げる MAP↓、組織灌流不足、SvO₂↓ → 戻すと改善する反応性を見る
③ 低体温の効果 temp を 30℃に下げ、flow も 1.8 に 低酸素消費下で「低灌流でも代償できる」現象を確認
④ 復温時の高需要 temp を 37 に戻し、flow は 2.0 のまま 代謝↑ vs 灌流不変 → SvO₂↓・乳酸↑ → flowを 2.4-2.6 に上げる必要
⑤ FiO₂単独調整の限界 SvO₂が下がった時、FiO₂のみ↑する 変化は小さい → 「酸素供給の根本は flow と Hb」を実感
