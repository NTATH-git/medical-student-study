1. HIT とは

HIT(Heparin-Induced Thrombocytopenia)は、ヘパリン投与に伴う 免疫介在性の血小板減少 で、特異な臨床像として「血小板が減るのに 出血ではなく血栓症が起こる」という矛盾性(paradoxical)の病態をきたします。

⚠ 死亡率: 未診断・未治療のHITは死亡率20-30%。血栓症(DVT, PE, 動脈血栓、心筋梗塞、脳梗塞、四肢虚血)の発症率は30-50%。

頻度

  • 未分画ヘパリン(UFH)使用例: 0.5-5%
  • 低分子量ヘパリン(LMWH)使用例: 0.1-1%(UFHより低い)
  • 心臓外科後: 1-3%(ヘパリン大量使用のため高め)
  • 内科系入院・透析患者: 比較的低頻度
💡 心血管外科で特に注意: CPB(人工心肺)では大量ヘパリンを使用 → 暴露歴がある患者の再手術時はHIT 急性発症のリスク。術前に血小板数推移を必ず確認。

2. Type 1 と Type 2 ― 似て非なるもの

「HIT」と総称されることもありますが、Type 1(非免疫性、軽症、予後良好)と Type 2(免疫介在性、重症、血栓症)は全く別の病態です。臨床的に「HIT」と言えば原則としてType 2を指します。

Type 1 (非免疫性)

  • 頻度: ヘパリン投与の10-30%(高頻度)
  • 機序: ヘパリンの直接的な血小板凝集作用
  • 時期: 投与開始 1-4日(早期)
  • 血小板: 軽度低下(>100,000/μL)
  • 血栓症: なし
  • 対応: ヘパリン継続でも自然回復

Type 2 (免疫介在性) ★臨床上の「HIT」

  • 頻度: 0.5-5%
  • 機序: 抗 PF4/heparin 抗体 → 血小板活性化 → 血栓
  • 時期: 投与5-14日目(典型)
  • 血小板: 中等度〜高度低下(>50%減 or <100,000/μL)
  • 血栓症: あり(動脈・静脈)
  • 対応: ヘパリン即時中止 + 代替抗凝固

3. 機序 ― なぜ血小板が減って血栓ができるのか

4ステップの病態

  1. ヘパリン投与 → PF4と結合
    血小板から放出される血小板第4因子(PF4)がヘパリンと複合体を形成
  2. PF4/heparin複合体に対する抗体(IgG)が産生される
    免疫系が複合体を「異物」と認識(5-14日かかる)
  3. 抗体-PF4-heparin複合体が血小板FcγRIIaに結合
    血小板が活性化 → 凝集 → 消費による血小板減少
  4. 活性化血小板から微小粒子放出・トロンビン産生
    過凝固状態 → 動脈・静脈血栓症

典型的な発症時期

  • Typical onset: ヘパリン開始 5-14日目(抗体産生に時間がかかる)
  • Rapid-onset HIT: 24時間以内に発症(過去100日以内にヘパリン暴露歴 → 抗体既存)
  • Delayed-onset HIT: ヘパリン中止後 9-40日に発症(まれ)

4. 4Ts スコア(臨床確率の評価)

HITの臨床診断には4Tsスコアを用います。4項目それぞれを 0-2点で評価、合計0-8点。

① Thrombocytopenia
(血小板減少)

2点: 50%以上減少 + 最低 ≥20,000
1点: 30-50%減少 or 10,000-19,000
0点: <30%減少 or <10,000

② Timing
(発症時期)

2点: 5-10日目 or 100日以内の暴露歴あり
1点: >10日 or 不明
0点: <4日(暴露歴なし)

③ Thrombosis
(血栓症)

2点: 新規血栓・皮膚壊死・アナフィラキシー
1点: 進行性 or 再発血栓 or 紅斑
0点: なし

④ oTher causes
(他原因の除外)

2点: 他原因なし
1点: 他原因の可能性あり
0点: 明確な他原因あり

スコア解釈

合計確率方針
6-8点高確率(>50%)即座にヘパリン中止 + 代替抗凝固 + 抗体検査
4-5点中確率(10-50%)原則 ヘパリン中止 + 抗体検査(結果待ち)
0-3点低確率(<5%)HIT以外を検索、ヘパリン継続可

5. 検査と確定診断

免疫学的検査(immunoassay)

  • 抗PF4/heparin抗体 ELISA: 感度高(>97%)・特異度低(~80%)。スクリーニングに使う
  • 陰性なら HIT を否定できる(NPV高)
  • 陽性でも他原因の偽陽性あり、機能的検査で確認

機能的検査(functional assay)

  • SRA(Serotonin Release Assay): ゴールドスタンダード。感度・特異度ともに高い
  • HIPA(Heparin-Induced Platelet Aggregation): 代替
  • 結果に数日かかるが、確定診断に必要
💡 検査結果を待たない: HITが疑わしければ 抗体検査の結果を待たずに ヘパリン中止 + 代替抗凝固を開始する。血栓症リスクが時間とともに増えるため。

6. 治療 ― やること・やらないこと

✅ やること

  1. すべてのヘパリンを即時中止(UFH, LMWH, ロック用ヘパリン、ヘパリン被覆カテーテル、ヘパリン入り輸液)
  2. 代替の抗凝固薬を開始(下記)
  3. 下肢血管エコーでDVTスクリーニング(無症状でも30%にDVT)
  4. 血小板正常化後(>150,000)にワーファリンへ移行(代替抗凝固と4-5日 overlap)
  5. カルテに「ヘパリン禁忌」を記録、患者にも情報提供

❌ やってはいけないこと

  • ヘパリンを「減量」して継続 ― 用量にかかわらず免疫反応は持続
  • LMWHへ変更 ― UFHと交差反応する(LMWH禁忌)
  • ワーファリン単独開始 ― 初期にプロテインC低下で warfarin-induced skin necrosis / venous limb gangrene リスク
  • 血小板輸血 ― 通常は禁忌(血栓助長)。出血合併症時のみ慎重に
  • 下大静脈フィルター挿入 ― 推奨されない

代替抗凝固薬(non-heparin anticoagulants)

薬剤分類投与経路備考
アルガトロバン直接トロンビン阻害(DTI)持続静注日本で第一選択。腎機能不全でも使用可、肝代謝
ビバリルジンDTI持続静注心臓外科CPBに用いる(本邦では適応外使用)
ダナパロイドヘパリノイド静注/皮下本邦でDIC適応。HITには適応外だが使用例あり
フォンダパリヌクスXa阻害薬皮下注本邦HIT適応外、海外では使用

7. 心臓外科患者でのHIT管理

心臓血管外科では人工心肺(CPB)で大量ヘパリン(300-400 U/kg)を投与するため、HIT既往または陽性患者の手術は特殊な対応が必要です。

術前HIT抗体評価

  • 過去のヘパリン暴露後 100日以内 + 抗体陽性 → 術前にplasmapheresis(血漿交換)で抗体除去 + ヘパリン使用
  • 100日以上経過 + 抗体陰性化 → ヘパリン使用可(再感作リスクは残る)
  • 強い陽性で延期不可 → ビバリルジンCPB(専門施設のみ)

ビバリルジン CPB の特徴

  • 半減期 25分と短い → CPB離脱後速やかに凝固復帰
  • ACT測定不可(凝固系の他指標でモニター)
  • ストック液による回路停滞は禁忌(血栓化リスク)
  • 術後出血リスクはヘパリン-プロタミン群と同等
📌 周術期の代替抗凝固: 術前1週間以上ヘパリン投与歴がある患者は、血小板トレンドを毎日確認。50%以上の低下があれば HIT 抗体検査 → 代替抗凝固に変更。
