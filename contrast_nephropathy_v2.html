1. なぜ心血管外科患者は腎機能が悪い人が多いのか

循環器内科・心臓血管外科の入院患者を診ていると、「またeGFRが30台か…」という場面が頻繁にあります。これは偶然ではなく、動脈硬化を共通基盤とする全身病態がいくつも重なっているためです。

① 動脈硬化は全身性疾患

心血管疾患の主要原因である動脈硬化は、冠動脈・脳動脈・腎動脈に同じ機序で進行する。心血管疾患の患者は 腎血管にも動脈硬化を抱えている。

② 共通リスク因子の集積

高齢・糖尿病・高血圧・脂質異常症・喫煙はすべて、心血管疾患と慢性腎臓病(CKD)の両方を進める。CVS患者ではほぼ全例で複数が併存。

③ 心腎症候群(cardio-renal syndrome)

心機能低下 → 腎灌流低下(forward failure)
右心不全 → 腎静脈うっ血(backward failure)。
慢性心不全患者の30-50%にCKDを合併。

④ 動脈硬化性腎血管病(RAS)

腎動脈狭窄(renal artery stenosis)は冠動脈疾患患者の20-30%に合併。RA系亢進 → 高血圧悪化 → さらなる腎障害の悪循環。

⑤ 累積的な腎障害

過去のPCI・CAG・CT・心臓手術で繰り返し腎臓に負荷(造影剤・低灌流・人工心肺)。「気がつけばcreatinine 1.5」状態。

⑥ 治療薬の影響

RAS阻害薬・利尿薬・NSAIDsは腎血流調節に影響。脱水・低血圧で腎機能が容易に悪化する状態にある。

⑦ 大動脈疾患特有の要因

腎動脈分岐近傍の 動脈瘤・解離・EVAR後 では腎動脈口の血流が直接影響を受ける。胸腹部大動脈瘤手術ではintraoperative AKIが高頻度。

⑧ 加齢そのもの

CVS手術患者の年齢中央値は70歳台後半。加齢に伴うnephron数減少と糸球体硬化で、若年と同じ手術侵襲でも腎機能が下がりやすい。

📌 結論: CVS患者の腎機能不良は「ひとつの原因」ではなく、動脈硬化を基盤に複数の要因が累積した結果。だから周術期管理では 「腎臓は既に弱っている」前提で動くことが重要。

2. 心腎症候群(Cardio-Renal Syndrome; CRS)― 5つのタイプ

Ronco分類(2008)によると、心臓と腎臓の相互作用は5つに分類されます。

Type 1
急性心腎症候群
急性心不全 → 急性腎障害(AKI)
Type 2
慢性心腎症候群
慢性心不全 → 慢性腎臓病(CKD)
★ CVS患者で最多
Type 3
急性腎心症候群
AKI → 急性心不全(肺水腫・不整脈)
Type 4
慢性腎心症候群
CKD → 慢性心血管病進行
Type 5
二次性心腎症候群
全身性疾患(敗血症・SLE等)による同時障害
💡 Type 2の機序: 心拍出量↓ → 腎灌流圧↓ → GFR↓ + RAS亢進 → 体液貯留 → 右心不全 → 腎静脈圧↑ → さらにGFR↓ という 負のスパイラル。NT-proBNPとeGFRが同時に悪化していく。

3. 造影剤腎症(CI-AKI)とは

定義

造影剤腎症(Contrast-Induced Acute Kidney Injury; CI-AKI)は、ヨード造影剤投与後 48-72時間以内 に、他に明らかな原因なく 血清クレアチニン値が 0.3 mg/dL以上 または 50%以上上昇 した状態と定義されます(KDIGO基準)。

頻度

  • 一般的にはCAG・CT造影後 1-5%
  • CKD合併例: 10-25%
  • 糖尿病 + CKD: 30-40%
  • PCI後・大量造影剤後: さらに高率

機序

  1. 髄質虚血: 造影剤による腎血管収縮(直接効果 + endothelin/adenosine放出)→ 髄質の酸素需給バランス悪化
  2. 直接細胞傷害: 尿細管上皮への酸化ストレス・ROS産生
  3. 浸透圧負荷: 高浸透圧造影剤は管内圧上昇 → tubular obstruction
  4. 粘性増加: 尿細管内造影剤が高粘度 → 流れ停滞
⚠ 予後: CI-AKI を発症すると、入院期間延長・心血管イベント増加・1年死亡率2-4倍(可逆性でも軌跡を悪化させる)。だから「予防」が最重要。

4. 生理食塩水輸液のメリット ― 5つの機序

造影検査前後の 等張生理食塩水(0.9% NaCl)輸液 は、最もエビデンスが確立したCI-AKI予防策です。なぜ効くのか、機序を5つに分けて整理します。

1髄質血流の改善

循環血液量↑ → 心拍出量↑ → 腎血流(特に低酸素になりやすい 髄質)の灌流改善。造影剤による血管収縮を血流量で打ち消す。

2造影剤の希釈

尿量増加で尿細管内の造影剤濃度↓ → 細胞毒性・浸透圧負荷の軽減。輸液していない場合の数倍に希釈される。

3造影剤の排泄促進

尿量↑ → 尿細管通過時間↓ → 造影剤の 滞留時間が短縮。粘性による流れ停滞を防止。

4RA系の抑制

循環血液量↑ → renin分泌↓ → アンギオテンシンII↓ → 腎血管収縮の緩和。RA系亢進による髄質虚血を抑える。

5尿pH・浸透圧の維持

適切な尿浸透圧維持により 酸化ストレスを軽減。重炭酸ナトリウム輸液も同様だが、近年は生食と差がないとの報告(PRESERVE試験)。

5. 標準的なプロトコル

💉 待機検査(CAG/CT/PCI)の場合

  • 輸液剤: 0.9% 生理食塩水(等張)
  • 速度: 1 mL/kg/h(LVEF低下例では 0.5 mL/kg/h)
  • 期間: 検査 6-12時間前から開始、検査後も 6-12時間継続(合計12-24時間)
  • 総投与量: 体重に応じ 1000-2000 mL程度
  • 外来検査ではより短期間プロトコル(検査前後1-3時間)も可

🚨 緊急検査(STEMIなど)の場合

  • 検査開始前に数百mL急速輸液(可能な範囲で)
  • 検査後は持続輸液6-12時間を必ず実施
  • 時間を理由に省略しない ― 心不全合併でない限り輸液は安全

輸液製剤の選択

7. 生食以外の予防策

  • 造影剤量の最小化: 必要十分量に留める。「使用量 < eGFR × 3」が一つの目安
  • 低浸透圧 or 等浸透圧造影剤を選択: 高浸透圧造影剤は避ける(国内ではほぼ使われない)
  • 腎毒性薬剤の一時中止: NSAIDs、メトホルミン(乳酸アシドーシス回避目的、48時間休薬推奨)、aminoglycosides
  • RA系阻害薬の一時休止: 症例により判断(完全な合意はない)
  • 段階的検査: 連日の造影検査を避ける(48時間以上空ける)
  • 術前 eGFR の評価とリスク説明: 同意取得時に CI-AKI リスクを明示
  • 非造影モダリティの活用: 非造影CT、MRI(ガドリニウムも腎機能注意)、エコー、近赤外などで代替検討
💡 心血管外科の現場でよくあるシナリオ: 「術前 CAG + 心臓CT + 大動脈CT」と立て続けに造影検査が予定された場合、可能なら 1日 1検査・間に48時間以上の輸液休息日を入れる。
