1. なぜこのネットワークを理解する必要があるか

腹部大動脈瘤の手術(EVAR、開腹)では、下腸間膜動脈(IMA)や内腸骨動脈を意図的に塞栓・結紮する場面があります。これらの動脈を「塞いでも大丈夫か」を判断する根拠が、側副血行ネットワーク(collateral circulation)です。

⚠ ネットワークが破綻すると…
  • 結腸虚血(ischemic colitis) — 死亡率高い、再開腹必要
  • 骨盤虚血 — 殿筋跛行(buttock claudication)、勃起障害
  • 脊髄虚血 — 対麻痺(まれ、Adamkiewicz動脈関連)
💡 大原則: 「血流は別経路で代償できるか」を術前CTで予測することが、これらの合併症を防ぐ唯一の方法。

2. 腸間膜動脈系(Mesenteric Arteries)

消化管を栄養する3本の主幹動脈:

3つの主幹

  • 腹腔動脈(CA: Celiac Artery) — T12レベル / 肝・脾・胃・十二指腸上部・膵頭
  • 上腸間膜動脈(SMA: Superior Mesenteric Artery) — L1 / 十二指腸下部〜横行結腸右2/3
  • 下腸間膜動脈(IMA: Inferior Mesenteric Artery) — L3 / 横行結腸左1/3〜直腸上部

IMAの3つの分枝

  • 左結腸動脈(left colic artery) — 下行結腸を支配
  • S状結腸動脈(sigmoid arteries) — S状結腸を支配
  • 上直腸動脈(superior rectal artery) — 直腸上部を支配
📌 ポイント: IMAはL3レベルから分岐するので、腎動脈下AAA(infrarenal AAA)の瘤腔内に開口することが多い。EVARで覆われる/結紮される可能性大。

3. 内腸骨動脈系(Internal Iliac Artery)

総腸骨動脈は仙腸関節の高さで 外腸骨動脈(EIA) と 内腸骨動脈(IIA, hypogastric artery) に分岐します。

内腸骨動脈の主な分枝

前枝(anterior division)▶
  • 閉鎖動脈(obturator)
  • 下殿動脈(inferior gluteal)
  • 内陰部動脈(internal pudendal)
  • 下膀胱動脈/子宮動脈(inferior vesical / uterine)
  • 中直腸動脈(middle rectal artery) ← 重要(IMA系との吻合)
後枝(posterior division)▶
  • 腸腰動脈(iliolumbar)
  • 外側仙骨動脈(lateral sacral)
  • 上殿動脈(superior gluteal)

下直腸動脈(inferior rectal artery)はどこから?

内陰部動脈の分枝で 直腸下部・肛門管 を支配。これも IMA系との吻合を形成する重要動脈。

4. 3つの主要側副血行路

① Marginal artery of Drummond(辺縁動脈)

SMAとIMAの末梢分枝が大腸辺縁で連結し、結腸全体を縁取る1本の連続した動脈アーケード。SMA-IMA間の最も恒常的な側副路。

  • SMA側: 中結腸動脈・右結腸動脈・回結腸動脈
  • IMA側: 左結腸動脈・S状結腸動脈
  • 連結点が脾湾曲付近 = Griffith point(脆弱)

② Arc of Riolan(リオラン弓)

中結腸動脈(SMA)と左結腸動脈(IMA)を結ぶ、腸間膜内側の太い側副路。通常は細いが、慢性的なIMA狭窄/閉塞があると拡張・蛇行して側副血行を担う ("meandering mesenteric artery")。

📌 CT読影: 太く蛇行したArc of Riolanが見えたら、IMA・腹腔動脈・SMAのいずれかに有意狭窄がある 強い示唆。この動脈は絶対に温存・結紮回避。

③ 直腸動脈経由(IMA ↔ 内腸骨動脈)

IMA系の上直腸動脈と、内腸骨動脈系の中直腸動脈・下直腸動脈が直腸壁内で吻合。骨盤と腸間膜の血流をつなぐ重要なバイパス。

  • IMAを塞いでも → 内腸骨動脈 → 中・下直腸動脈 → 上直腸動脈 → 左結腸動脈 と逆流で代償可
  • 逆に内腸骨動脈を塞いでも → IMA → 上直腸動脈 → 中・下直腸動脈 と順行で代償可

Watershed area(脆弱領域)

名称部位支配動脈の境界
Griffith point脾湾曲(splenic flexure)SMA(中結腸動脈)と IMA(左結腸動脈)の境界
Sudeck critical point直腸S状結腸移行部(S3)S状結腸動脈最末梢と上直腸動脈の境界

これらの領域は側副血行が乏しく、IMA結紮後の虚血性大腸炎の好発部位(特に脾湾曲)。

6. 臨床的意義(血管外科のシナリオ)

シナリオ1: EVAR で IMA が瘤腔内開口している

大半は 結紮(覆い)てよい(無治療でも辺縁動脈 + Riolan弓 + 直腸吻合で代償される)。

確認すべき: ① SMA・腹腔動脈に有意狭窄がない / ② 内腸骨動脈が両側開存 / ③ IMA径が4mm以上で太い場合は注意 / ④ 過去の腸管手術で側副が破壊されていない

シナリオ2: AAA が腸骨動脈に及び 内腸骨動脈をプラグする必要

片側塞栓なら通常は耐えられるが、両側塞栓は禁忌(殿筋虚血、骨盤虚血、勃起障害)。

対策: 内腸骨動脈温存型デバイス(IBE: Iliac Branch Endoprosthesis)を選択。または bell-bottom 法、サンドイッチ法

シナリオ3: CT で Riolan弓が太く蛇行している

IMA、SMA、または腹腔動脈のいずれかに有意狭窄がある証拠。

対策: Riolan弓は絶対に温存。瘤覆いがRiolan弓を含む場合は再建を検討。IMA結紮で結腸虚血の高リスク

シナリオ4: 開腹AAA手術中 IMA を結紮する判断

術中evaluation: IMA結紮前に クランプテスト(数分IMA遮断)→ S状結腸の色調、ドップラー血流、結腸辺縁拍動を確認 → 良好なら結紮、不良なら IMA再建(S状結腸動脈に吻合)

背景: AAA開腹手術後の虚血性大腸炎は1-3%、死亡率50%以上の重篤合併症
⚠ 結腸虚血の早期警告: 術後の血性下痢・腹部膨満・代謝性アシドーシス・WBC↑/CRP↑は虚血性大腸炎の典型像。即座にS状結腸鏡(下部消化管内視鏡)で評価。
