1. なぜ重要か ― 対麻痺は最も恐れられる合併症

Gray's Anatomy(1918)の大動脈図 ― 上行・弓部・下行大動脈と3分枝(腕頭・左総頸・左鎖骨下)。
画像出典:Henry Gray Anatomy of the Human Body Plate 506 ― Wikimedia Commons (Public Domain)

胸部下行大動脈瘤(DTAA)・胸腹部大動脈瘤(TAAA)に対する人工血管置換術の最重大合併症が 対麻痺(paraplegia) です。下肢の永続的麻痺は患者のQOLを劇的に低下させ、社会復帰を不可能にします。

⚠ 発生頻度:
・胸部下行大動脈瘤(DTAA)単独置換: 2-8%
・胸腹部大動脈瘤(TAAA)置換: 4-30%(Crawford II型は最高リスク)
・急性大動脈解離 緊急手術: さらに高率

関連する用語の区別

  • Paraplegia(完全対麻痺): 下肢完全麻痺
  • Paraparesis(不全対麻痺): 下肢筋力低下
  • Immediate onset: 術中〜術直後発症 = 術中の脊髄虚血
  • Delayed onset: 術後12-48時間で発症 = reperfusion injury + 二次的虚血
💡 重要: Delayed paraplegia は「術後一旦動かせた下肢が、12-48時間後に動かなくなる」典型像。観察が継続必要。

2. 脊髄血流の解剖

脊髄の動脈血流は、1本の前脊髄動脈(ASA)と左右の後脊髄動脈(PSA)から供給されます。前脊髄動脈は脊髄前2/3(運動野・側索)を支配するため、虚血すると運動麻痺が前景に立ちます。

3階層の血流網

  1. 椎骨動脈系: 頚髄〜上位胸髄を栄養
  2. 分節動脈(segmental artery): 胸部・腰部大動脈から肋間動脈/腰動脈として分岐 → 各椎間孔から radicular artery として脊髄前動脈に流入
  3. 内腸骨動脈経由: 仙髄・円錐部分の側副血行

Adamkiewicz 動脈(大根動脈)

分節動脈の中で最も太い脊髄供給枝を Adamkiewicz動脈(arteria radicularis magna) と呼びます。

  • 分岐レベル: 75%が T8-L1、特にT9-T12(左側からが多い)
  • 胸髄下部〜腰膨大部の前脊髄動脈の 主要供給源
  • 個人差が大きく、術前CTで描出されないこともある
  • TAAA手術中にこの動脈を 遮断・結紮 すれば、対麻痺リスクが急増
前脊髄動脈(運動) 後脊髄動脈(感覚) 分節動脈 Adamkiewicz動脈

側副血行ネットワーク(collateral network)

近年は単一のAdamkiewicz動脈ではなく、椎骨動脈・分節動脈・内腸骨動脈経由でつながる広範な血管ネットワークとして脊髄血流を理解する概念(Etz, Griepp)が提唱されています。これにより、複数の分節動脈をゆっくり遮断していけば、ネットワーク全体で代償する余地があります。詳しくは次セクション参照。

3. 脊髄血流の理論的理解 ― ダム理論と側副ネットワーク

脊髄虚血を防ぐためには「個々の動脈」よりも「血流ネットワーク全体」を理解することが鍵です。本邦・海外で提唱されてきた代表的な2つの概念を紹介します。

① ダム理論(河原田 修義 教授・札幌医科大学 心臓血管外科)

脊髄血流を 「複数の流入源(支流) + 共通の貯水池(side branch network)」 として捉える概念です。貯水池(=椎骨周囲血管網, paraspinal collateral network)に流入する血流ソースが複数ある限り、一部の流入を絶っても水位(=灌流圧)は保たれます。

機能している流入路 遮断された流入路 貯水池(灌流圧) 虚血閾値

臨床的含意(ダム理論から導かれる戦略)

  • 分節動脈を一度に多数遮断しない ― 段階的遮断(sequential clamping)で貯水池の貯留能力に時間を与える
  • 他の流入を確保する ― 椎骨動脈・内腸骨動脈系を温存(両側内腸骨遮断回避)
  • 流入総和が閾値を下回ると一気に対麻痺 ― 個々のリスク評価より「総流入の予備能」を重視
  • 術前にAAA歴・TEVAR歴・骨盤血流低下があれば「貯水池の容量が既に小さい」と考える
💡 ダム理論が示す本質: 「Adamkiewicz動脈が温存できたか」より、「ネットワーク全体に十分な流入が確保されているか」が予後を決める。

② 椎谷 紀彦 教授(浜松医科大学)の側副ネットワーク図

椎谷教授は 脊髄を取り囲む椎旁・椎前血管網(paraspinal / paravertebral collateral network) の重要性を一連の研究と図解で広めました。脊髄前動脈に加えて、椎旁筋肉内・椎体周囲に張り巡らされた密な血管網が 「圧の安定化装置」 として働くという概念です。

主幹動脈・分節動脈 椎旁血管網 椎骨動脈系 内腸骨動脈系

臨床的含意(椎旁血管網の発達を促す戦略)

  • 左鎖骨下動脈・内腸骨動脈の温存 ― ネットワークの入口を保つ
  • 段階的TEVAR(staged repair) ― 期間を空けて2回に分けることで椎旁網が代償的に拡張する時間を与える
  • preoperative arterial conditioning ― 術前の段階的coil塞栓で椎旁網を発達させる(海外の試み)
  • 分節動脈の温存より、ネットワーク全体への入口確保が重要

2つの概念は補完的

📌 統合的理解:
・ダム理論(河原田): 量的概念 ― 流入路が複数あるから1本失っても水位は維持される
・側副ネットワーク(椎谷, Etz, Griepp): 構造的概念 ― 椎旁・椎前に密な吻合網があるから圧が均一化される
どちらも「Adamkiewicz単独依存」から「ネットワーク全体」への発想転換を促し、現代の脊髄保護戦略(staged repair, 内腸骨温存, distal aortic perfusion等)の理論的支柱となっている。

4. 対麻痺をきたす4つのメカニズム

1分節動脈の遮断

人工血管置換で 肋間動脈・腰動脈・Adamkiewicz動脈 を一括結紮 → 前脊髄動脈への灌流喪失 → 直接的な脊髄虚血。

2大動脈遮断による低灌流

胸部大動脈遮断中は 遠位灌流圧が低下(時に20-30mmHg)。distal aortic perfusion がなければ 脊髄供給動脈圧 ↓ → 灌流不足。

3塞栓(atheroembolism)

大動脈壁の 動脈硬化プラーク・血栓・コレステロール結晶 が分節動脈に飛散 → 微小血管閉塞。慢性解離で多い。

4再灌流障害・浮腫

大動脈遮断解除で脊髄灌流再開 → 虚血再灌流障害 + 細胞外浮腫 → CSF圧↑ → 脊髄灌流圧↓ → 二次的虚血(delayed paraplegia の主因)。

📌 SCPP(脊髄灌流圧)の公式: SCPP = MAP − CSFP
(MAP: 平均動脈圧、CSFP: 髄液圧)
分子(MAP)が下がる + 分母的に増える(CSFP)→ SCPP低下 → 虚血。
→ MAP を上げる(昇圧)+ CSFP を下げる(CSFドレナージ)が予防の根幹。
詳しくは「CSFドレナージ」のページへ。

5. Crawford 分類(胸腹部大動脈瘤の範囲)

瘤の範囲が広いほど、結紮する分節動脈数が多くなる = 対麻痺リスク上昇。

Type I
左鎖骨下動脈直下〜腎動脈上まで
対麻痺: 約 3-10%
Type II ★最高リスク
左鎖骨下動脈直下〜腸骨動脈分岐部まで(全長)
対麻痺: 約 10-30%
Type III
第6肋間以遠〜腸骨動脈分岐部まで
対麻痺: 約 5-15%
Type IV
横隔膜下〜腸骨動脈分岐部まで(腹部大動脈中心)
対麻痺: 約 1-5%
Type V (extent V)
第6肋間以遠〜腎動脈上まで
対麻痺: 約 5-10%
💡 なぜType IIが最も危険か: Type IIは 左鎖骨下動脈直下から腸骨動脈分岐まで という最大範囲で人工血管に置換するため、Adamkiewicz動脈を含めほぼ全ての分節動脈が結紮される。

その他のリスク因子

  • 緊急手術(待機手術より2-3倍リスク)
  • 大動脈遮断時間が長い(>30分)
  • 術前 脊髄虚血既往
  • 術前腎不全(MAP維持困難)
  • 高齢(>70歳)
  • 左鎖骨下動脈の被覆(TEVAR時)
  • 内腸骨動脈の閉塞・既往AAA術後 → 側副血行の予備能低下

6. 早期発見の方法

術中モニタリング

  • MEP(運動誘発電位): 経頭蓋電気刺激 → 下肢筋電図記録。脊髄前角細胞・側索を反映。50%以上の振幅低下で警告 → 緊急に分節動脈再建 or 灌流改善
  • SEP(体性感覚誘発電位): 後索を反映。前脊髄動脈領域の虚血では出にくいが、補助的に使用
  • 遠位灌流圧モニター: 大腿動脈ライン圧で distal aortic perfusion 圧を測定

術後評価

  • 麻酔覚醒後、可及的早期に下肢運動・感覚を評価
  • 覚醒できない症例(挿管継続)では 覚醒テスト(wake-up test) を朝行う
  • 術後12-48時間は最も警戒 — delayed paraplegia の好発時期
  • 下肢麻痺発症時は即座に MAP を上げ、CSFドレナージ強化、画像評価
⚠ Delayed paraplegia の対応: 動かせていた下肢が動かなくなったら、即座にCOCKPIT(MAP↑ + CSFP↓ + 酸素化↑ + 適切なヘマトクリット) プロトコルを発動。早期介入で回復例あり。

7. 予防戦略 ― 多角的アプローチ

単一の方法では効果限定。複数の戦略を組み合わせる(multimodal protection)のが標準。

TEVAR(ステントグラフト)時の追加配慮

  • 左鎖骨下動脈を被覆する場合は 事前バイパス(carotid-subclavian bypass) が推奨されることが多い
  • 過去のAAA手術歴があれば、内腸骨動脈経由の側副が低下 → CSFドレナージの閾値を下げる
  • 大動脈被覆範囲が長い場合(>20cm)は対麻痺リスク上昇
