1. エンドリーク(Endoleak)とは

エンドリークとは、EVAR(Endovascular Aortic Repair)やTEVAR後に、本来であれば瘤内へ流入が遮断されるはずの血流が 瘤腔(aneurysm sac)内に持続的に流入する状態 を指します。

📌 重要: エンドリークがあると瘤腔内圧が維持され、瘤径の拡大・破裂リスクが残るため、フォローアップで早期発見・対応が必須です。

分類の歴史

1996年に White らが Type I-IV を提唱、後に Type V(endotension)が加わり、現在は Type I-V の5タイプ に分類されます。タイプによって 緊急度・治療方針・予後 が大きく異なります。

EVAR後フォローアップ頻度

  • 術後1ヶ月、6ヶ月、12ヶ月で造影CT(CTA)を撮影
  • 以降は年1回のCTA(もしくは超音波エコーで代替)
  • 瘤径が増大していれば原因(エンドリーク)を精査

2. エンドリークの5タイプ

タイプごとに 原因部位 · 頻度 · 緊急度 が異なります。色分けで覚えましょう。

Type I

固定部からの漏れ

頻度: 約 5-10% / 緊急度: 高

ステントグラフトと自己血管の固定部(sealing zone)からの漏れ。瘤腔内圧が高くなり破裂リスク大。

  • Ia: 中枢側(proximal、腎動脈下大動脈)
  • Ib: 末梢側(distal、総腸骨動脈)
  • Ic: 内腸骨動脈プラグ部
治療: 緊急対応必須。バルーン拡張、cuff/extension追加、coil塞栓、コンバージョン(開腹手術)
Type II

側枝からの逆行性流入 ★最多

頻度: 約 20-40% / 緊急度: 中

瘤壁から分岐する側枝(腰動脈、下腸間膜動脈)からの逆行性血流。最も頻度が高い。

  • IIa: 1本の側枝から流入
  • IIb: 2本以上の側枝から流入(out-flow あり)
治療: 瘤径拡大なければ経過観察(自然消失あり)。拡大例はcoil塞栓(SMA経由のIMA、または直接穿刺)
Type III

グラフトの接合部・破損

頻度: 約 1-3% / 緊急度: 高

モジュラー接合部の離開(IIIa)または布部分の損傷・破損(IIIb)。瘤腔への直接流入で高圧。

治療: 緊急対応。追加のステントグラフト挿入で被覆、または開腹コンバージョン
Type IV

グラフトの透過性

頻度: 稀(術直後のみ) / 緊急度: 低

ステントグラフトの布(fabric)の多孔性からの一過性漏出。術直後の造影で確認。

治療: 通常は自然消失。ACT正常化で改善。経過観察のみで対応可
Type V

エンドテンション(endotension)

頻度: 稀 / 緊急度: 中

明らかなリークは検出されないが瘤径が拡大する状態。微小漏出 or 血栓化部位経由の圧伝播が原因と推定。

治療: 瘤径増大が顕著なら追加ステント、開腹コンバージョン考慮
💡 暗記のコツ: Type I = 入口/出口の隙間、Type II = 側枝逆流、Type III = グラフトの故障、Type IV = 布の汗、Type V = 原因不明の膨張 と覚えると整理しやすい。

4. 診断

第一選択: 造影CT(CTA)

  • 動脈相 + 遅延相(60-120秒)を撮影 → 遅延相でType IIが描出されやすい
  • 瘤腔内の造影剤プールがエンドリークの証拠
  • 流入経路(シール部 or 側枝 or グラフト接合部)からタイプ推定

代替/補助モダリティ

  • 造影超音波(CEUS): 腎機能低下例で被曝・造影剤を減らせる
  • MRA: ステント artifact が少ないNitinol製で有用
  • DSA(血管造影): 治療直結時の最終確認・治療計画

瘤径測定のポイント

  • 最大短径(short-axis diameter)で経時変化を追う
  • 年5mm以上の拡大 → エンドリーク精査・治療検討
  • 瘤径不変または縮小 → エンドリーク許容できる場合あり(特にType II)

5. タイプ別 治療方針

Type原則主な治療
Ia即対応Balloon angioplasty → Aortic cuff → Chimney/Snorkel/FEVAR/開腹コンバージョン
Ib即対応Iliac extension → 内腸骨動脈塞栓 + 総腸骨延長
Ic計画的追加塞栓 / extension
II瘤径拡大例のみSMA→IMA経由 coil塞栓 / 腰動脈trans-lumbar塞栓 / 経皮的sac穿刺塞栓
III即対応追加グラフト/relining(被覆) / 開腹コンバージョン
IV経過観察通常 自然消失
V瘤径拡大例のみ追加ステント / 開腹コンバージョン
⚠ 治療の閾値: Type II であっても 瘤径が年5mm以上拡大 または 絶対径が5.5cm超 なら治療介入を検討。経過観察に流されない。

用語意味
