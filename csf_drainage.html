1. なぜ効くのか ― SCPPの式から考える

対麻痺予防の本質は 「脊髄に十分な血液を流し続ける」 ことです。脊髄に流れる血液の圧(脊髄灌流圧、SCPP)は次の式で表されます。

SCPP = MAP − CSFP
脊髄灌流圧 = 平均動脈圧 − 脳脊髄液圧

つまり、脊髄を流れる血流の駆動圧は 「流入圧 − 流出抵抗」 の構造になっています。流出側の 脳脊髄液圧(CSFP) は、脊髄を取り囲む硬膜内の圧で、ここが高いと血管が圧迫されて流入血流が低下します。

大動脈瘤手術中・術後に起こること

❌ 介入しない場合

  • MAP↓ 大動脈遮断で遠位灌流低下
  • CSFP↑ 再灌流障害・脊髄浮腫で髄液圧上昇
  • SCPP = MAP↓ − CSFP↑ → 急激低下
  • → 脊髄虚血 → 対麻痺

⭕ CSFドレナージ + 昇圧

  • MAP↑ 昇圧薬で >80-90mmHg維持
  • CSFP↓ ドレナージで <10mmHg維持
  • SCPP = MAP↑ − CSFP↓ → 確保
  • → 脊髄灌流維持 → 対麻痺予防
💡 物理学的アナロジー: 脊髄血流を「ストローでジュースを吸う」と例えれば、CSFPは「ストローを外から指で押さえる力」。指の力(CSFP)が強いとジュース(血流)が流れない。CSFドレナージは「指を緩める」操作。

2. エビデンス ― Coselli 試験

Coselli ら 2002年 RCT(J Vasc Surg)

  • 対象: 胸腹部大動脈瘤 Crawford I/II型 待機手術 145例
  • 介入群: 周術期 CSFドレナージ実施 / 対照群: 実施せず
  • 結果: 対麻痺/不全対麻痺の発生率を80%相対減少(13% → 2.6%)
  • これを根拠に SVS・ESVS・JSVS ガイドライン はCrawford II型などの高リスクTAAAで Class I 推奨
📌 主要ガイドラインの推奨(要約):
・SVS Guidelines on Aortic Aneurysm: high-risk TAAA で CSFドレナージを Class I 推奨
・JSVS 大動脈瘤・大動脈解離診療ガイドライン: Crawford I/II/III型で考慮
・TEVAR でも 大動脈被覆範囲が広い場合 / 既往AAA術後 / 左鎖骨下被覆例で考慮

近年の知見

  • 術前予防的留置 vs 術後 rescue 留置 → 予防的留置の方が有効(術後発症してから入れても遅いことが多い)
  • 合併症リスクとのバランスは継続して議論中(後述)
  • TEVAR 単独例での予防的留置は controversial(routine ではない)

3. 適応と禁忌

✅ 適応(予防的留置)

  • Crawford II型 TAAA(全例推奨)
  • Crawford I・III型 TAAA(個別判断、原則推奨)
  • 広範囲 TEVAR(>20cm被覆)
  • 左鎖骨下動脈被覆を伴う TEVAR
  • 既往 AAA術後 / 既往 TEVAR後の追加TEVAR
  • 内腸骨動脈閉塞・両側骨盤虚血の既往

❌ 禁忌・要注意

  • 凝固障害(INR>1.5、血小板<100,000)
  • 抗凝固薬服用中(ワーファリン、DOAC)
  • 髄膜炎・脊椎部位感染
  • 頭蓋内圧亢進(出血、占拠性病変)
  • 解剖学的に穿刺困難(脊椎変形)
  • 意識障害で評価困難な場合は注意
📌 タイミング: 通常 麻酔導入後すぐ、手術開始前 にカテーテルを留置。術中・術後 48-72時間継続する。

4. 手技

カテーテル留置

  1. 体位: 側臥位または座位 で背中を丸める(腰椎穿刺と同じ)
  2. 穿刺部位: L3-L4 または L4-L5 椎間 を清潔操作で穿刺(脊髄円錐より下方)
  3. 使用カテーテル: 14-16G Tuohy 針 → 髄液逆流を確認 → 軟性カテーテル(EDM)を頭側に5-10cm挿入
  4. 固定・接続: カテーテルを皮下トンネル経由でしっかり固定(自己抜去防止)→ 閉鎖式ドレナージ回路に接続
  5. 圧トランスデューサ接続: CSFP モニタリング開始(基準点は外耳道高位)

ドレナージ管理(術中・術後)

  • 目標 CSFP: < 10 mmHg(8 mmHg以下が安全マージン)
  • ドレナージ速度上限: 10-15 mL/h(過剰排液 → 硬膜下血腫リスク)
  • 1日上限: 通常 150-250 mL/日 程度
  • 排液は 清澄(clear) が正常。血性なら出血、混濁なら感染を疑う
  • 留置期間: 通常 術後 48-72時間。リスク高ければ最大5日まで延長

抜去のタイミング

  • 神経学的に下肢運動正常を48時間以上維持できていれば抜去検討
  • 抜去前に 凝固機能を再確認(術後出血リスク管理のため一時的に抗凝固投与している場合あり)
  • 抜去後 24時間は仰臥位安静、頭低位は避ける

6. 合併症 ― リスクとのバランス

CSFドレナージは有効だが、無視できない合併症があるため 適応の慎重な吟味 が必要です。

合併症頻度機序・対策
硬膜下血腫1-3%過剰ドレナージ → 脳が下垂 → 架橋静脈裂傷。ドレナージ速度上限を遵守
頭痛(post-LP headache)5-15%CSF漏出による低髄液圧症状。安静・水分・カフェイン・blood patch
カテーテル関連感染(髄膜炎)0.5-2%清潔操作、長期留置を避ける
脊髄硬膜外血腫<1%凝固障害・抗凝固併用時。麻痺出現で疑い、緊急MRI
カテーテル離断・残存稀抜去時に抵抗あれば中止、画像評価
偽性麻痺・誤判定—抗凝固併用例で出血性発症と紛らわしい
⚠ 硬膜下血腫の警告サイン: CSFドレナージ後の意識障害・頭痛増悪・神経学的悪化があれば即頭部CTを実施。発症すれば mortality 高い。

7. 他の脊髄保護戦略との組み合わせ

CSFドレナージは単独使用ではなく、対麻痺予防の multimodal protection の中核 として位置付けられます。詳細は対麻痺メカニズムのページを参照。

戦略SCPP式での意味
CSFドレナージCSFP ↓(分母を下げる)
MAP維持(>80-90 mmHg)MAP ↑(分子を上げる)
Distal aortic perfusion大動脈遮断中も遠位灌流圧確保 → 分節動脈圧維持
分節動脈再建そもそも血流入口を保つ
軽度低体温虚血耐性向上(同じ低灌流でも被害少)
MEP/SEPモニター早期警告 → 上記戦略を即座に強化
💡 COCKPIT プロトコル(救済時): 術後に下肢麻痺発症 → 即座に CSF排液強化(目標 <5mmHg)+ MAP↑(>100 mmHg)+ 酸素化↑(SpO2 >95%)+ 適切なヘマトクリット(30%以上)で介入。早期対応で回復例あり。
