このサイトについて

心血管外科を中心に、医学生が実習・臨床現場で出会う疾患・手技・周術期管理のポイントを、図解とインタラクティブなクイズで学べる教材を集めたサイトです。スマートフォンからも快適に閲覧できます。各ページにはクイズと用語集を備え、5〜10分の隙間時間で理解度を確認できます。

🫀 心血管外科 総論

心血管外科 総論

解剖、主要疾患(虚血性/弁膜症/大動脈/先天性)、代表術式(CABG/AVR/TAVI/EVAR等)、周術期管理を網羅。8問クイズ付き。

全学年入門

心血管の圧 ― 高いところから低いところへ

大動脈(最高圧)と大静脈(最低圧)の対比、圧勾配が血流の動力という大原則、各部位の正常圧、視覚バー、臨床応用。7問クイズ。

全学年生理

🪡 心臓手術手技

正中切開 大動脈弁置換術(AVR)の手順

術前準備から閉胸まで7フェーズで解説。タイムライン、各ステップの注意点、TEE活用、合併症対策。7問クイズ付き。

高学年手技

手術器械(心血管外科)図鑑

基本器械(持針器/剪刀/鑷子) + 血管遮断鉗子(Satinsky/DeBakey/Bulldog/Rumel) + 心臓特有器械 + 縫合糸の使い分け。6問クイズ。

全学年器械

ガウンテクニック(マスク・滅菌ガウン装着)

手指消毒(WHO推奨ラビング法)、マスク・帽子、closed glove technique、滅菌野ルール、脱ガウンまで。6問クイズ。

全学年OR基礎

🫶 補助循環(MCS: Mechanical Circulatory Support)

IABP(大動脈内バルーンパンピング)

最も歴史が長い補助循環。拡張期増強(冠血流↑)+ 後負荷軽減の2大効果。タイミング異常・離脱・IABP-SHOCK II試験。7問クイズ。

全学年MCS

VA-ECMO(静脈-動脈ECMO)

心肺ともに代行する究極の補助循環。回路・カニュレーション・Harlequin症候群・LV unloading・ECPR・離脱。7問クイズ。

高学年MCS

VV-ECMO(静脈-静脈ECMO)

肺機能のみ代行。重症ARDS・COVID-19の最終手段。Avalon Elite、ultra-protective ventilation、recirculation、sweep gas離脱。7問クイズ。

高学年MCS

Impella(経皮的補助人工心臓)

カテーテル先端のマイクロ軸流ポンプ。LV → 大動脈に順行性拍出+LV unloading。モデル別、ECPELLA、suction events。7問クイズ。

高学年MCS

⚙️ 体外循環(CPB)・心筋保護

人工心肺(CPB)の基本

構成要素(貯血槽・ポンプ・人工肺・熱交換器)・回路の流れ・運転管理(灌流量・温度・ACT)・合併症(SIRS, AKI, 神経学的)。6問クイズ。

全学年CPB

心筋保護の基本

高K心停止の機序、結晶質/血液/del Nido cardioplegia、順行性/逆行性投与、warm hot shot、再灌流障害。6問クイズ。

高学年CPB

CPBシミュレーター(インタラクティブ)

スライダーで灌流量・体温・Hct・FiO₂を操作し、MAP/SvO₂/DO₂I/乳酸の変化を体験。5フェーズ・5シナリオ。

高学年JS体験

🩺 血管外科 ― 動脈瘤と血流ネットワーク

EVAR術後 エンドリークの種類と対応

Type I-Vの5タイプの機序・頻度・緊急度・治療を整理。模式図とタイプ別治療方針表付き。6問クイズ。

全学年血管外科

下腸間膜動脈・内腸骨動脈の血流ネットワーク

Drummond辺縁動脈・Arc of Riolan・直腸吻合の側副路。AAA手術での結紮可否判断・虚血性大腸炎予防。6問クイズ。

全学年解剖

胸部・胸腹部大動脈瘤術後の対麻痺メカニズム

脊髄血流解剖(Adamkiewicz動脈)・SCPP式・4つの機序・Crawford分類・予防戦略。7問クイズ。

高学年脊髄

CSFドレナージが対麻痺に有効な理由

SCPP = MAP − CSFPの式で理解する。Coselli RCTの根拠、適応・禁忌、手技、合併症(硬膜下血腫)。7問クイズ。

高学年脊髄保護

Loeys-Dietz症候群(LDS)

TGF-β経路の遺伝子変異による結合組織疾患。古典的三徴、マルファンとの違い、早期手術閾値(4.0-4.4 cm)、全身モニタリング。7問クイズ。

高学年遺伝

💊 周術期管理 ― 抜去手技・陰圧・凝固

CV(中心静脈)カテーテル抜去の注意点

致死的合併症である空気塞栓を防ぐための知識と手技。体位・呼吸法・閉塞性ドレッシング。PMDA No.71の実例 + 7問クイズ。

全学年手技

ドレーン抜去時に気をつけること

胸腔/縦隔/腹腔/皮下の部位別 抜去手技と合併症。PMDA No.60・No.62の事例 + 7問クイズ。

全学年手技

持続陰圧療法(NPWT/VAC)はなぜ有効か

創傷治癒を加速する4つの作用機序、適応・禁忌、心臓外科後縦隔炎での応用。6問クイズ。

全学年創傷

HIT(ヘパリン誘発性血小板減少症)

Type 1/2の違い、PF4抗体の機序、4Tsスコア、代替抗凝固薬、心臓外科でのビバリルジンCPB対応。7問クイズ。

高学年血液

心血管外科患者の腎機能と造影剤腎症予防

なぜCVS患者は腎機能が悪いのか(8つの理由)、心腎症候群、CI-AKIの機序、生理食塩水輸液5つの機序、Mehranスコア。7問クイズ。

全学年腎保護

血液透析(HD)の基礎

なぜ必要か(腎機能4つ)、何をしているか(拡散・UF・対流)、週3回×4時間の意味、シャント・AVF/AVG/CVC、HDとCRRTの比較表。8問クイズ。

全学年透析

放射線防御の三原則

距離(逆二乗の法則)・時間(パルス透視・低線量)・遮蔽(鉛PPE)。CVS現場での実践、水晶体線量厳格化(2020)、確率的/確定的影響。8問クイズ。

全学年安全

抗凝固薬の周術期管理

ワーファリン/UFH/LMWH/DOAC/抗血小板薬 ― 術前休薬・ブリッジング・緊急時中和(イダルシズマブ/アンデキサネット)・術後再開。6問クイズ。

高学年抗凝固

📚 リファレンス・基礎

身体の中で陰圧になるところはどこか

胸腔・気道・心室・関節腔・食道・中耳など各部位の陰圧の生理。CV抜去/NPWT/NPPEなど臨床応用と医療事故。6問クイズ。

全学年生理

心血管外科 用語集(英日対訳)

200+項目の英日対訳・略語集。検索機能 + カテゴリフィルタ(解剖/疾患/術式/機器/薬物/検査/CPB/合併症)。
