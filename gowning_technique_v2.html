1. なぜガウンテクニックが重要か

手術室では SSI(surgical site infection、手術部位感染) 予防のために、術者の手指・体表からの細菌侵入を防ぐ厳密な手順があります。心血管外科では胸骨縦隔炎(死亡率10-25%)・人工弁・人工血管感染など、感染合併症が致死的になりうるため、ガウンテクニックは特に重要です。

⚠ 学生が起こしがちな汚染:
  • 手を下げる・後ろを振り向く → ガウン背中(非滅菌)を術野に触れさせる
  • マスクを触ったまま手洗い直後にガウンへ → マスクに付着した細菌を手袋へ
  • ガウン装着前に器械台に触る → 触れた瞬間に滅菌野汚染
  • 顎マスクで会話 → 飛沫が術野へ

滅菌野の3つの原則

  1. 滅菌物どうしのみ接触可(非滅菌に触れた瞬間に汚染)
  2. 視線の届かない場所は非滅菌扱い(背中、腰より下、テーブル端より外側)
  3. 濡れた = 汚染(液体は細菌の通り道になる ― strikethrough contamination)

3. 入室前の準備

  1. 更衣: スクラブ(術衣)に着替え、上は下に入れる。靴も手術室専用
  2. 装飾品の除去: 指輪・腕時計・ネックレス・ピアスは原則外す(細菌温床)
  3. 爪: 短く切り、マニキュア・ジェル禁止
  4. 傷の確認: 手指に傷・湿疹があれば申告(防水絆創膏)
  5. 長髪: 髪を完全に帽子内に収める
💡 トイレは入室前に: 手洗い後にトイレに行けば全てやり直し。

4. マスク・帽子の装着

順序(必ずこの順)

  1. 帽子装着: 髪を完全に内側に。眉毛上・耳をしっかり覆う
  2. マスク装着: 上下を確認(ノーズワイヤー側が上)→ 紐をかける(上の紐:後頭部、下の紐:首)→ ノーズワイヤーを鼻形に密着
  3. マスクが密着しているか確認: 隙間から空気漏れがないことを確認

マスクの種類

種類用途備考
サージカルマスク(三層)標準。通常手術飛沫(>5μm)を防ぐ
シールド付きマスク大量出血予想、血液・体液飛散症例結膜を保護
N95マスク結核・空気感染対策、感染症例密着度試験(fit test)必要
⚠ マスクの取り扱い:
  • 手洗い後はマスクに絶対触れない(マスク表面は非滅菌)
  • 「顎マスク」(マスクを顎に下げる)は禁止 ― 細菌付着 + 飛沫拡散
  • 濡れたら交換(濡れたマスクは細菌が通り抜ける)
  • 1症例ごとに交換が原則

5. 手指消毒(scrub)― 3つの方法

方法A:従来法(ブラッシング法、現在は推奨度低下)

  • 消毒石鹸(ヒビスクラブ・イソジン)+ ブラシ
  • 各指 各面を10ストロークずつ計5-10分
  • 頻回はブラッシングで皮膚バリア破壊 → 近年は推奨されない

方法B:短縮スクラブ(現在の標準)

  • 消毒石鹸で 2-5分 手指洗浄(ブラシは爪の周囲だけ)
  • 肘より下を流水で 指先側から肘側へ 一方向にすすぐ(汚染水が指先に逆流しないため)
  • 滅菌タオルで指先 → 肘の順に拭く

方法C:ラビング法(アルコール擦込み法、現在推奨)

  • 事前に 通常の手洗い(石鹸 + 流水)で目に見える汚れを除去
  • アルコール製剤(クロルヘキシジン含有グルコネート)を 3 mL 程度
  • 手指 → 前腕 → 肘へ 計2-3回(規定の手順を1.5-3分)
  • 乾燥するまで擦り続ける(乾く前に手を下げない)
  • 水・タオル不要、皮膚バリアへの負担最小
💡 WHO推奨: アルコールラビング法が最も効果的かつ皮膚刺激が少ない。施設での標準化が進む。

共通の禁忌動作

  • 手を 肘より下に下げない(水・消毒液が指先側に流れて再汚染)
  • 手洗い後に 顔・髪・服に触れない
  • 濡れた手で何かに触れたら再洗浄

6. ガウン装着

ガウンの構造を理解する

  • 滅菌部分: 前面(胸〜腰)、腕の肘から手首、上半身
  • 非滅菌扱い部分: 背中、腰より下、肩より上、袖の手首より先(=手袋でカバー)

装着手順(自分で着る場合)

  1. 滅菌ガウンを受け取る: 看護師が滅菌パックを開き、ガウンの内側を上にしてテーブルに広げる
  2. 襟を持つ: ガウンの内側(襟付近)を両手で掴む。外側(=滅菌面)には絶対触れない
  3. 展開: ガウンを身体の前で展開、胸より高い位置で広げる(床に触れさせない)
  4. 両腕を通す: 同時に両腕をスリーブに通す。手は袖の中に留めて先端を出さない(closed glove technique用)
  5. 看護師による後ろ留め: 看護師が背中側からガウンの首・腰紐を結ぶ(術者は前を向いたまま)
  6. 腰紐の交換(自分が回す): 手袋装着後、腰紐の前面を非滅菌の看護師に渡す → 自分が回って受け取る(360°回転で背中側からカバー)
📌 closed glove technique: ガウンの袖から手を出さずに、袖越しに手袋を引き上げる方法。手袋装着までガウン外面の滅菌が完全に保たれる。心血管外科で標準。

7. 手袋装着 ― Closed Glove Technique

標準的手順

  1. 右手用手袋を取る: ガウン袖の中の左手で、滅菌手袋パックから右手用手袋を取り出す
  2. 手のひら側を上に: 右手袋の 掌側を自分の前腕上に乗せ、指先が肘側を向くように配置
  3. 手袋の折返しを掴む: 袖の中の左手で手袋折返し(カフ)の 下層(肌側)を掴む
  4. 右手で手袋を被せる: 右手側の袖から指を出しながら、手袋を被せる
  5. 左手も同様に: 装着済みの右手(滅菌)で左手袋の 外側カフ を持ち、左手に被せる
  6. カフを伸ばす: 両手袋のカフがガウンスリーブをしっかり覆うよう伸ばす

二重手袋(double gloving)

  • 心血管外科・大手術では 二重装着 が標準
  • 外側手袋の損傷・穿孔リスク低減(針刺し事故時の血液暴露減)
  • 内側は色付き(例:緑)を使うと穿孔検出しやすい(indicator gloves)

8. 滅菌野ルール ― やっていい/ダメな例

✅ やっていい

  • 胸の前で両手を組む(滅菌領域内)
  • 滅菌された器械・ガウンに触れる
  • 術野の上方を移動する
  • テーブル端より内側に手を置く
  • 胸〜腰の高さに手を保つ

❌ やってはいけない

  • 手を腰より下に下げる(=非滅菌扱い)
  • 背中側に手を回す
  • マスク・帽子・自分のガウン襟を触る
  • 非滅菌の人・物に触る(壁、麻酔器、モニター)
  • 顔を掻く、汗を拭う
  • 滅菌ガウン同士でも背中側で触れ合う

術野内移動の基本

  • 表表で交差: 滅菌どうしのすれ違いは前後に向き合って
  • 裏裏で交差: 非滅菌どうしは背中合わせで
  • 180°回転は危険: 背中(非滅菌)が術野を通過する
⚠ 学生がやりがちな汚染と対処: 「あっ、汚染した!」と思ったらすぐ申告。気づかないふりが最大の害悪。看護師がガウン/手袋を交換してくれる(ペナルティではない)。

9. 脱ガウンの手順

術後の脱ガウンは、体表・術野・器械を汚染させない順序で行います。

  1. 腰紐を切る/外す: 自分で前で切る or 看護師に頼む
  2. ガウンを脱ぐ: 首と肩の部分を引き、外側(汚染面)が内側になるように丸めながら脱ぐ
  3. 手袋装着のままガウンを廃棄: 手袋は最後
  4. 手袋を脱ぐ: 内側を出すように外側を掴んで脱ぐ → 反対手も同様
  5. マスク・帽子は最後: 紐のみを持って外す(マスク表面に触れない)
  6. 手指消毒: アルコール擦込み
💡 順序の理由: ガウン → 手袋 → マスク の順で外せば、最も汚染リスクの高い手袋を最後まで保持できる。
