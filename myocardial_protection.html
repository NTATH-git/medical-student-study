1. なぜ心筋保護が必要か

心臓手術中は大動脈遮断(cross-clamp)で冠血流を止め、心臓を停止させて手術します。この間 心筋は虚血状態 に陥ります。何もしなければ心筋細胞は壊死してしまうため、心筋保護(myocardial protection) が必須です。

心筋保護の目的

  • 心停止の確実な誘導(電気的活動を即座に停止)
  • 代謝の最大限抑制(虚血耐性を高める)
  • 細胞内Ca²⁺過剰の防止(reperfusion injury予防)
  • 心筋構造の保護(浮腫・酸化ストレス低減)
  • 大動脈遮断解除後の心機能回復(stunned myocardiumの最小化)
⚠ 心筋保護失敗の代償: 不十分な心筋保護 →術後Low Output Syndrome・致死的不整脈・心筋梗塞。場合により補助循環(IABP, ECMO)が必要に。

2. 心筋保護液(Cardioplegia)の作用機序

核となる原理:高K⁺による脱分極停止

正常心筋細胞の静止膜電位は約 −90 mV。高K⁺(15-25 mEq/L) を流すと細胞外K↑ → 膜電位が脱分極(−50 mV付近)で停止 → Na⁺チャネル不活化 → 活動電位発生不能 → 電気的心停止。これは「diastolic arrest(拡張期停止)」と呼ばれます。

追加の保護要素(成分による)

  • 低体温化: 心臓表面冷却 + 心停止液自体を4-10℃ → 代謝50%削減 / 10℃低下
  • Mg²⁺: Caチャネル拮抗、虚血障害軽減
  • 緩衝剤(THAM, HCO₃⁻): 虚血性アシドーシス抑制
  • マンニトール: 浮腫予防・フリーラジカル捕捉
  • 糖質(グルコース): 嫌気的代謝の基質提供
  • L-アスパラギン酸/グルタミン酸: ATP産生回復促進(Buckberg)

3. 心停止液の主な種類

① 結晶質心筋保護液(Crystalloid)

電解質溶液型。代表:St. Thomas溶液、HTK(Custodiol)。

特徴: 単純で安価、酸素運搬なし。HTKは1回大量(2L以上)で長時間保護可。

② 血液心筋保護液(Blood Cardioplegia)

CPBの酸素化血液+電解質を混合。Buckberg法(4:1血液:晶質)が代表。

特徴: 酸素運搬・緩衝能あり、生理的。冷血/温血、間欠/持続など多様。日本で最多。

③ del Nido心筋保護液

小児用に開発、近年成人にも普及。低Ca・Mg高・リドカイン含有。

特徴: 1回投与で60-90分保護。投与間隔が長く、手術中断が少ない。MICS・短時間手術で重宝。

4. 投与経路

① 順行性(antegrade)

  • 大動脈基部 → 冠動脈口 → 冠循環
  • 大動脈遮断後、大動脈基部の心筋保護液カニューレから注入
  • 生理的な経路、最も標準
  • AR(大動脈弁閉鎖不全)では効果不十分 → 大動脈切開後に直接冠動脈口注入 or 逆行性併用

② 逆行性(retrograde)

  • 冠静脈洞 → 心筋静脈 → 毛細血管 → 動脈側へ逆流
  • 冠静脈洞カニューレを右房経由で挿入
  • AR例・重症冠動脈狭窄例で順行性を補完
  • 注入圧 30-50 mmHg(高圧は冠静脈洞損傷リスク)

③ 冠動脈口直接注入

  • 大動脈切開後、左右冠動脈口に直接 hand-held cannula で注入
  • 大動脈基部置換(Bentall)・重症AR例で確実
💡 順行 + 逆行 併用: 順行で得られない領域(LADの慢性閉塞末梢など)も逆行で灌流できる。重要症例で「順行 → 逆行 → 順行」のサンドイッチ投与が一般的。

5. 投与プロトコル

典型的なBlood Cardioplegia(Buckberg)プロトコル

  1. 導入(induction): 大動脈遮断直後、冷血液心筋保護液 1000-1500 mL 順行性に注入 → 数秒で電気的心停止
  2. 維持(maintenance): 20-30分ごとに 300-500 mL 反復投与(冷却・脱K洗い出し防止)
  3. 再灌流前(reperfusion): 大動脈遮断解除前に「温血終末投与(warm hot shot)」 ― 温血(37℃)+ アスパラギン酸/グルタミン酸でATP回復促進
  4. 遮断解除: 心拍再開(自然 or 除細動 10-30J)

心筋温度管理

  • 心筋温度プローブで 10-15℃ を目安に維持
  • 心臓表面に 氷スラッシュ(slush) や冷生食を散布
  • 右心系は冷却されにくい(温暖化リスク) → 心筋保護液投与・心表面冷却の継続

大動脈遮断時間と心筋障害

  • 適切な保護下では 60-90分以内 は通常安全
  • 2時間超で reperfusion injury リスク上昇
  • 3時間超は要注意 → cardioplegia 高頻度投与、温血終末強化
⚠ Reperfusion injury: 遮断解除直後の急激な酸素・Ca流入で活性酸素・Ca過剰 → 細胞障害。Warm hot shot や緩徐な再灌流(controlled reperfusion)で軽減を図る。
