1. 心血管外科とは

心血管外科(Cardiovascular Surgery; CVS)は、心臓・大血管(大動脈・主要肺動脈)・末梢血管の外科的治療を担う診療科です。冠動脈バイパス術(CABG)、弁膜症手術、大動脈瘤手術、先天性心疾患手術などを扱い、近年は低侵襲化(MICS・TAVI・EVAR/TEVAR)とハイブリッド治療が急速に進歩しています。

💡 学習のコツ: 心血管外科は「解剖→病態生理→診断→治療選択→術式」の流れで理解すると体系的に頭に入ります。まず正常解剖と血流の方向をしっかり押さえましょう。

2. 心臓・大血管の解剖

下図は心臓と主要大血管の模式図です。各部位をタップして名称を確認できます。

動脈血 静脈血 肺動脈(静脈血) 房室弁 半月弁

血流の順序(覚え方)

全身 → 上下大静脈 → 右房 → (三尖弁) → 右室 → (肺動脈弁) → 肺動脈 → 肺 → 肺静脈 → 左房 → (僧帽弁) → 左室 → (大動脈弁) → 大動脈 → 全身

📌 ポイント: 心臓弁膜症の手術名は「弁名 + 術式」で命名されます。
例: AVR = Aortic Valve Replacement(大動脈弁置換術) / MVP = Mitral Valve Plasty(僧帽弁形成術)

3. 主要疾患(タブで切替)

虚血性心疾患(Ischemic Heart Disease)

冠動脈の狭窄・閉塞により心筋への血流が低下し、狭心症や心筋梗塞をきたす疾患群。

  • 原因: 動脈硬化(高血圧・脂質異常症・糖尿病・喫煙が主要リスク)
  • 症状: 胸部絞扼感・労作時呼吸困難・冷汗・放散痛(左肩・下顎)
  • 診断: 心電図、心エコー、冠動脈CT、冠動脈造影(CAG)
  • 治療: 薬物 → PCI(カテーテル) → CABG(冠動脈バイパス術)
    • 左主幹部病変・3枝病変・糖尿病合併ではCABGが優位(SYNTAX試験)

心臓弁膜症(Valvular Heart Disease)

4つの弁(AV/MV/TV/PV)のいずれかが狭窄または逆流をきたす疾患。

疾患主な原因代表術式
AS(大動脈弁狭窄症)加齢性石灰化、二尖弁AVR / TAVI
AR(大動脈弁閉鎖不全症)弁尖変性、大動脈基部拡張AVR / 自己弁温存(David)
MS(僧帽弁狭窄症)リウマチ性MVR / PTMC
MR(僧帽弁閉鎖不全症)変性(Barlow)、虚血性MVP(形成) / MVR
TR(三尖弁閉鎖不全症)機能性(肺高血圧)TAP(三尖弁形成)
💡 重要: 僧帽弁逆流(MR)は可能な限り形成術(MVP)を選択。耐久性・抗凝固不要・術後LV機能保持で置換術(MVR)より優位。

大動脈疾患(Aortic Disease)

  • 胸部大動脈瘤(TAA): 上行・弓部・下行に分類。径5.5cm以上で手術適応。
  • 腹部大動脈瘤(AAA): 腎動脈下が大半。5.0-5.5cm以上、または年間5mm以上拡大で手術。
  • 大動脈解離(Aortic Dissection): Stanford分類が重要
    • A型(上行大動脈を含む): 緊急手術適応(死亡率1%/時間)
    • B型(上行を含まない): 原則 降圧薬による保存的治療、合併症あれば TEVAR

術式: 人工血管置換術(開胸/開腹) / ステントグラフト(TEVAR/EVAR) / Hybrid(両方の組み合わせ)

先天性心疾患(Congenital Heart Disease; CHD)

出生児の約1%に発生。チアノーゼの有無で大別される。

  • 非チアノーゼ性(左→右シャント): ASD, VSD, PDA, AVSD
  • チアノーゼ性: Fallot四徴症(TOF), 大血管転位症(TGA), 単心室症, 総肺静脈還流異常(TAPVR)

新生児・乳児期から段階的手術(BTシャント→Glenn→Fontan)が行われることが多い。

4. 代表的な術式

カードをタップ(またはクリック)で詳細展開。

CABG

Coronary Artery Bypass Grafting

狭窄冠動脈の遠位側に内胸動脈(LITA)や橈骨動脈、大伏在静脈をバイパス。

開心術

AVR

Aortic Valve Replacement

大動脈弁を機械弁または生体弁で置換。ASに対する標準術式。

開心術

TAVI

Transcatheter Aortic Valve Implantation

カテーテルで生体弁を留置。高齢・ハイリスクASに対する低侵襲治療。

カテーテル

MVP / MVR

Mitral Valve Plasty / Replacement

MRに対し形成(MVP)を第一選択。困難例・MS は置換(MVR)。

開心術

EVAR

Endovascular Aortic Repair

AAAに対するステントグラフト内挿術。鼠径部小切開で実施。

血管内

TEVAR

Thoracic Endovascular Aortic Repair

胸部大動脈瘤・B型解離に対するステントグラフト。

血管内

Bentall

Bentall Operation

大動脈基部置換 + AVR + 冠動脈再建。マルファン症候群などで実施。

開心術

David

Valve-Sparing Root Replacement

大動脈弁を温存して基部を置換。若年・弁形態良好例で選択。

開心術

体外循環(CPB)の基本

人工心肺(Cardiopulmonary Bypass)とは ▶

右房から脱血 → 人工肺で酸素化 → 上行大動脈に送血することで、心臓を止めて手術を可能にする装置。

  • カニュレーション部位: 上行大動脈(送血) + 上下大静脈または右房(脱血)
  • 心筋保護: 冷却 + 心筋保護液(心停止液; cardioplegia)で心停止と心筋保護を達成
  • 合併症: 全身炎症反応、凝固障害、神経学的合併症など
Off-pump CABG(OPCAB)とは ▶

人工心肺を使わず心拍動下にCABGを行う術式。スタビライザーで吻合部位を固定。脳合併症が少ない一方、グラフトの開存性は施設依存。

MICS(低侵襲心臓手術) ▶

Minimally Invasive Cardiac Surgery。胸骨正中切開を避け、右小開胸などで弁手術やCABGを行う。整容性・早期回復が利点。

5. 周術期管理のポイント

術前評価

  • 心機能(EF, 弁機能, 肺動脈圧)、冠動脈評価(CAG/CCTA)
  • 呼吸機能(FEV1)、腎機能(eGFR, Cre)、肝機能
  • 頸動脈エコー(脳梗塞リスク)、歯科コンサルト(感染源)
  • リスクスコア: EuroSCORE II, STS Score, JapanSCORE

術中

  • TEE(経食道心エコー)による弁機能・心機能のリアルタイム評価
  • ヘパリン化(ACT > 400-480秒)・人工心肺管理
  • 心筋保護(順行性/逆行性 cardioplegia)

術後

  • ICU管理(血行動態、出血、不整脈、呼吸)
  • 抗凝固/抗血小板療法(機械弁ではワーファリン生涯)
  • 心臓リハビリテーション(早期離床)
⚠ 主要合併症: 出血・再開胸(2-5%)、心房細動(20-40%)、脳梗塞(1-3%)、AKI(5-30%)、縦隔炎(0.5-2%)、Low Output Syndrome
