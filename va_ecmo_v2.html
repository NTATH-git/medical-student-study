VA-ECMO
(静脈-動脈ECMO)

1. VA-ECMO とは

人工心肺(CPB)の基本回路(BruceBlaus / Blausen Medical)
① 静脈脱血 → ② 貯血槽 → ③ ポンプ → ④ 人工肺・熱交換器 → ⑤ 動脈送血。 心血管手術中、心臓と肺の役割を装置が一時的に肩代わりする。
画像出典:BruceBlaus ― Wikimedia Commons (CC BY 3.0)

VA-ECMO(Veno-Arterial Extracorporeal Membrane Oxygenation、静脈-動脈ECMO)は、静脈から脱血し、人工肺で酸素化した血液を動脈へ送血する体外循環装置です。心臓と肺の両方を 完全に代行 できるため、心原性ショック・心停止に対する最強の補助循環として用いられます。

3行サマリ

  • 機構: 静脈血を抜き → 遠心ポンプ → 人工肺(O₂↑/CO₂↓)→ 動脈に送血(平均 3-5 L/min)
  • 適応: 心原性ショック・劇症型心筋炎・PE・ECPR(心停止)・post-cardiotomy
  • VS CPB: 開心術用の CPBより簡略化(リザーバーなし・閉鎖回路)、ベッドサイドで数日〜2週間運転可
💡 ECMO の "MO" は ガス交換装置: ECMO = "Extracorporeal Membrane Oxygenation"。膜型人工肺(membrane oxygenator)を使う体外循環全般を指す。VA(静脈-動脈)と VV(静脈-静脈)があり、VAは心肺サポート、VVは肺サポートに特化(VV-ECMOページ参照)。

2. 回路と構成要素

① 脱血カニューレ(venous)

drainage

大腿静脈 → IVC・右房に留置(21-25Fr)。径が大きいほど大きい流量が得られる。

② 遠心ポンプ(centrifugal)

駆動

回転数(rpm)で流量制御。前負荷・後負荷の影響を受ける(CPBのローラーポンプと異なり 非定容量)。

③ 膜型人工肺(oxygenator)

ガス交換

中空糸膜で O₂供給・CO₂排出。sweep gas flow でCO₂、FiO₂ でPaO₂を調節。長期使用で膜劣化(血漿リーク)。

④ 送血カニューレ(arterial)

return

大腿動脈 → 下行大動脈に留置(15-21Fr)。径が大きいほど血流障害↑ → 下肢虚血リスク↑

⑤ 熱交換器

体温管理

oxygenatorに内蔵。低体温療法(33-36℃)併用可、ECPR後の脳保護に。

⑥ 送血側分岐(distal perfusion)

下肢保護

大腿動脈からの送血だと末梢(下肢)が虚血になるため、順行性灌流ライン を浅大腿動脈や前脛骨動脈に追加することが多い。

3. カニュレーション(挿入部位)

末梢挿入(peripheral V-A)― 標準

  • 脱血: 右(or左)大腿静脈 → IVC → 右房入口部
  • 送血: 反対側(or 同側)大腿動脈 → 下行大動脈へ 逆行性 に送血
  • 利点: 切開不要・ベッドサイドで挿入可・ECPRに最適
  • 欠点: 下肢虚血、北南症候群(後述)、逆行性灌流による LV圧負荷

中枢挿入(central V-A)

  • 脱血: 右房 / 上下大静脈
  • 送血: 上行大動脈 → 順行性 に送血
  • 適応: post-cardiotomy ECMO(胸骨開放のまま装着)、CPB 離脱困難症例
  • 利点: 流量大、北南症候群なし
  • 欠点: 開胸必須・出血リスク高い・感染リスク
⚠ 北南(North-South / Harlequin)症候群: 末梢VA-ECMOで自己心拍が回復してくると、上半身は自己心肺由来の(換気不良で低酸素な)血液、下半身はECMO由来の高酸素血になり、頭部・冠動脈が低酸素に陥る現象。右橈骨動脈SpO₂と下肢SpO₂の解離で検出。対策は呼吸器設定強化、VAV-ECMOへの変更、Impella併用など。

4. 適応と禁忌

主な適応

  • 急性心筋梗塞後 心原性ショック(refractory cardiogenic shock)
  • 劇症型心筋炎(fulminant myocarditis)― 高い回復率(>70%)で良好な適応
  • 急性肺塞栓症(massive PE)― 血栓溶解・摘除術の橋渡し
  • 偶発性低体温症(復温と心肺サポート)
  • ECPR(extracorporeal CPR)― 心停止に対する補助循環付き蘇生
  • Post-cardiotomy shock(CPB離脱困難)
  • 心移植・LVAD への橋渡し(bridge to decision/transplant/durable VAD)
  • 移植後 primary graft dysfunction

禁忌(絶対 / 相対)

絶対禁忌相対禁忌
  • 不可逆的脳損傷
  • 進行末期癌
  • 不可逆的多臓器不全
  • 重症大動脈弁逆流(AR)
    ※逆行性送血が逆流を悪化
  • 急性大動脈解離(末梢挿入)
  • 高度PAD(末梢挿入困難)
  • 制御困難な出血
  • 免疫不全(感染リスク)
  • 高齢(>75歳目安)+ 著明な併存症
  • 心停止時間 > 60分(ECPR)
📊 ECPR の時間軸: 心停止から ECMO 確立までの "low-flow time" は 60分以内 が目安。これを超えると脳・多臓器障害が不可逆になる確率が急上昇。Bystander CPRの質、現場搬送、ER体制が予後を決める。

5. 管理のポイント

主要パラメータ

項目目標値備考
ECMO流量3-5 L/min(60-80 mL/kg/min)体表面積比 2.0-2.4 L/min/m²
MAP60-80 mmHg低すぎは灌流不足、高すぎは LV後負荷↑
SvO₂≥ 70%組織酸素需給バランスの指標
PaO₂(動脈)80-100 mmHg過剰なら高酸素血症リスク
右橈骨動脈SpO₂監視必須上半身灌流の質(北南症候群検出)
ACT160-200秒CPB(>400秒)より低い
APTT40-60秒ヘパリン持続。HIT既往はアルガトロバン
Hb / Hct7-10 / 25-30%輸血閾値は施設により異なる
血小板≥ 50,000機械的破壊で減少
遊離Hb・LDH監視溶血の指標

呼吸器設定(ECMO中のlung rest)

  • 低圧・低換気: Pplat ≤ 25, PEEP 10-15, FiO₂ 0.3-0.5, RR 10-12
  • VA-ECMOでも肺は使うため(特に自己心拍回復時)、適切な PEEP で無気肺予防

抗凝固管理

  • 導入時: ヘパリン 50-100 U/kg ボーラス、その後持続(ACT/APTTで滴定)
  • HIT 既往: アルガトロバン(0.1-0.5 μg/kg/min)
  • 出血リスク高い場合: 抗凝固を一時中断し低用量で運転(ただし血栓化リスク↑)

7. LV unloading(左室減圧)― VA-ECMOの最大課題

なぜLV過伸展が問題か

末梢VA-ECMOは大腿動脈から逆行性に送血するため、大動脈圧(=LV後負荷)が著明に上昇 します。一方、衰弱した左室は十分に拍出できず:

  1. LV内に血液が滞留 → LV拡張・過伸展
  2. 左房圧↑ → 肺水腫
  3. LV内血栓形成
  4. 心筋酸素需要↑ → 回復遅延

LV unloading の方法

方法機序備考
Impella 併用(ECPELLA / ECMELLA)LV → 大動脈にポンプ送血最も効果的だが、Impella合併症(溶血)+ コスト・経験
IABP 併用後負荷軽減効果は限定的だが手技容易・安価
外科的 LV ventドレナージLV内血液を直接吸引開心術中、または胸骨開放下に
左房 vent(心房中隔穿刺)左房からカテーテルで吸引カテ室で経心房中隔的に
循環作動薬(ドブタミン等)LV収縮力↑で駆出させる低用量で。過剰投与は心筋酸素需要↑
💡 LV unloading のタイミング: 大動脈弁が開かない(動脈圧波形が「平ら」)、肺水腫、LV拡張 → 早期に unloading 戦略を発動する。「待ち」は禁物。

8. 離脱(weaning)

離脱基準(turn-down test)

  • 原疾患の改善(MI再灌流、心筋炎回復、肺塞栓血栓除去後など)
  • ECMO流量を 1.5 L/min 程度まで段階的に減量 し、血行動態が維持されるか
  • TEE で LV駆出量・大動脈弁開放の確認
  • MAP ≥ 60mmHg、CI ≥ 2.2、SvO₂ ≥ 65%、乳酸正常
  • 低用量カテコラミン(ノルアド ≤ 0.1 μg/kg/min 程度)

カニューレ抜去

  • 末梢挿入: 経皮的に圧迫止血 or 外科的縫合(動脈側は通常切開・修復が必要)
  • 中枢挿入: 開胸・直視下に動静脈を修復
  • 抜去後はパテンシー確認(下肢・腎・脳の灌流)
📝 「橋渡し戦略」を初期から: ECMOは "bridge to..." の装置。導入時から (1)回復(recovery)、(2)決断(decision)、(3)移植・durable VAD、(4)取り外し(withdrawal) のどこに向かうかを 多職種カンファ で議論しておく。
